# PvP vs PvE 분리 설계서

## 개요
동일한 캐릭터/스킬 시스템에서 PvP와 PvE 환경별로 밸런스를 분리하여
각 모드에서 다양한 메타가 형성되도록 한다.

## 분리 적용 영역

### 1. 수치 보정 계수

| 항목 | PvE | PvP |
|---|---|---|
| 힐 효율 | 100% | 70% (-30%) |
| 보호막 흡수량 | 100% | 60% (-40%) |
| CC 지속턴 | 원본 | -1턴 (최소 1턴) |
| CC 적중률 | 원본 | -15% |
| DoT 대미지 | 100% | 80% (-20%) |
| 즉사 HP 기준 | 25% | 15% |
| 속도비례딜 상한 | 200 | 170 |
| 디버프→버프 취급 | 최대 2개 | 최대 1개 |

### 2. 스킬별 PvP 보정

#### 회복 계열
- 모든 힐 스킬: PvP에서 회복량 30% 감소
- 이유: PvP에서 힐러 의존도가 지나치게 높아지면 경기 시간 과다

#### CC 계열
- 빙결/수면/석화: PvP에서 지속 -1턴, 적중률 -15%
- 이유: Hard CC가 PvP에서 턴 박탈 → 체감 불쾌도 높음
- 침묵: PvP에서 지속 -1턴만 적용 (적중률 유지)

#### 즉사 계열
- 아누비스 즉사: PvE 25% → PvP 15%
- 이유: PvP에서 HP 25% 이하 즉사는 탱커까지 위협

#### 보호막 계열
- 모든 보호막: PvP에서 흡수량 40% 감소
- 이유: M08 보호막연합이 PvP에서 무적 상태 → 경기 교착

#### 속도 계열
- 속도비례 대미지: PvP 상한 170 (PvE 200)
- 속도 버프 중첩: PvP에서 최대 3회 중첩 제한

### 3. PvE 전용 메커니즘

#### 보스전 특수 룰
- 보스는 CC 내성 보유 (Hard CC 확률 50% 감소)
- 보스는 즉사 면역
- 보스는 넉백 면역
- 보스 분노 게이지: 일정 피해 누적 시 강화 패턴

#### 스테이지별 속성 보정
- 화속성 스테이지: 화속성 적 방어력 +30%, 수속성 아군 대미지 +20%
- 수속성 스테이지: 수속성 적 속도 +20, 목속성 아군 대미지 +20%
- 등등 각 속성별 스테이지 보정

#### 웨이브 시스템
- 3~5웨이브 구성
- 웨이브 간 SP 유지, HP/보호막 유지
- 쿨타임은 웨이브 전환 시 초기화

### 4. PvP 전용 메커니즘

#### 밴픽 시스템
- 각 플레이어 1캐릭터 밴 (상대 사용 불가)
- 선밴 후밴 교대 → 전략적 밴 필요

#### 턴 제한
- 최대 15턴 제한
- 15턴 초과 시 생존 HP% 합산으로 승패 결정
- 철벽수호팀의 무한 지구전 방지

#### 속도 동률 처리
- PvP에서 동일 속도 시: 공격형 > 마법형 > 서폿 > 힐러 > 탱커 순
- 같은 직업이면 성급 높은 쪽 우선

#### 랭킹 시스템
- 시즌제 (4주 단위)
- 시즌별 밴 캐릭터 자동 지정 (S티어 팀 핵심 캐릭터 1명)
- 시즌 보상: 한정 스킨, 강화 재화

### 5. PvP/PvE 메타 차이 예상

#### PvE 메타 예상
| 티어 | 팀 | 이유 |
|---|---|---|
| S | M01 화상연소 | 보스전 DoT 축적 극강, CC 불필요 |
| S | M04 치명타학살 | 잡몹 즉사로 웨이브 빠른 클리어 |
| A | M05 속도압도 | 멀티히트로 웨이브 속도 클리어 |
| A | M09 디버프착취 | 보스 디버프 축적 → 일격 |
| B | M06 철벽수호 | 안전하지만 클리어 속도 느림 |

#### PvP 메타 예상
| 티어 | 팀 | 이유 |
|---|---|---|
| S | M06 철벽수호 | 턴제한 내 방어력으로 HP% 승리 |
| S | M09 디버프착취 | 철벽수호 메타 카운터 |
| A | M02 빙결제어 | CC 약화되지만 속도감소는 유효 |
| A | M05 속도압도 | 선공 확보 → 빠른 킬 |
| B | M08 보호막연합 | 보호막 40% 너프로 하락 |
| B | M01 화상연소 | DoT 20% 너프 + 힐로 상쇄 |

## 구현 우선순위
1. PvP 수치 보정 계수 시스템 (config 기반)
2. PvP 턴 제한 + HP% 판정
3. PvE 보스 CC 내성
4. PvE 웨이브 시스템
5. 밴픽 시스템

## 스킬 상세

### 노멀
- 설명: 적 1명 대미지
- 강화+1: 대미지 +10%
- 강화+2: 적중 시 아군 1명 소량 보호막

### 액티브
- 설명: 체력이 낮은 아군 1명 보호막 + 버프 지속 1턴 증가
- 강화+1: 보호막 수치 +15%
- 강화+2: 보호막 중인 아군 추가로 공격력 +10%

### 얼티밋
- 설명: 아군 전체 보호막 + 디버프 1개 제거
- 강화+1: 제거 개수 +1 추가
- 강화+2: 버프 중인 아군 수만큼 전체 버프 지속 +1턴

### 패시브
- 설명: 보호막이 있는 아군이 받는 디버프 지속 1턴 감소
- 강화+1: 보호막 수치 +15%
- 강화+2: 보호막 아군의 버프 지속 +1턴

## 속성 상성
- 유리: 암속성
- 불리: 없음속성

## 핵심 포인트
- 강점: 팀 버프·시너지 — 보호막
- 약점: 낮은 단독 전투력 — 아군 없이 무력

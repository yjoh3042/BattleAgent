## 스킬 상세

### 노멀
- 설명: 적 1명 대미지
- 강화+1: 피격 시 자힐 5%
- 강화+2: 적중 시 30% 확률로 도발

### 액티브
- 설명: 자신 반격확률 증가
- 강화+1: 반격 확률 +15%
- 강화+2: 반격 시 50% 확률로 대상 침묵 부여

### 얼티밋
- 설명: 전열의 적 대미지 + 방어력 감소
- 강화+1: 대미지 +20%
- 강화+2: 도발 대상 공격 시 받는 대미지 흡혈 30%

### 패시브
- 설명: 도발 중인 대상 공격시 추가 대미지
- 강화+1: 추가 대미지 수치 +15%
- 강화+2: 도발 대상 처치 시 자신 HP 10% 회복

## 속성 상성
- 유리: 광속성
- 불리: 없음속성

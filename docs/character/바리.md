# 바리 — 스킬 컨셉 상세 기획서

## 캐릭터 아이덴티티
| 항목 | 값 |
|---|---|
| 속성 | Water |
| 직업 | Magician |
| 성급 | 3★ |

## 스킬 상세

### 노멀
- 설명: 적 1명 대미지
- 강화+1: 대미지 +10%
- 강화+2: 30% 확률로 속도 감소

### 액티브
- 설명: 적 1명 4타 대미지 + 속도 감소
- 강화+1: 타격 횟수 +1
- 강화+2: 적중 시 50% 확률로 빙결 1턴

### 얼티밋
- 설명: 적 전체 대미지 + 적 전체 속도 감소
- 강화+1: 대미지 +20%
- 강화+2: 속도가 감소된 적에게 추가 대미지 +30%

### 패시브
- 설명: 속도가 자신보다 느린 적을 공격시 50% 확률로 빙결
- 강화+1: 발동 확률 50% → 65%
- 강화+2: 빙결 부여 시 대상 속도 추가 감소

## 속성 상성
- 유리: 화염속성
- 불리: 목속성

## 핵심 포인트
- 강점: 광역 CC/디버프 — 빙결, 속도, 전체 대미지
- 약점: 낮은 HP/DEF — 선공을 잡히면 위험

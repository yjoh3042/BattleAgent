## 스킬 상세

### 노멀
- 설명: 적 1명 대미지
- 강화+1: 피격 시 자힐 5%
- 강화+2: 적중 시 자신에게 소량 보호막

### 액티브
- 설명: 자신 보호막 부여 + 전열의 적 도발
- 강화+1: 도발 지속 +1턴
- 강화+2: 도발 중 받는 대미지 -10%

### 얼티밋
- 설명: 아군 전체 보호막 + 자신 방어력 증가
- 강화+1: 보호막 수치 +20%
- 강화+2: 보호막이 깨지지 않은 아군 공격력 +15% 2턴

### 패시브
- 설명: 보호막이 깨질 때 자신 주변 아군 방어력 증가
- 강화+1: 방어력 증가 수치 +15%
- 강화+2: 보호막 파괴 시 자신 방어력 +15% 2턴

## 속성 상성
- 유리: 암속성
- 불리: 없음속성

## 핵심 포인트
- 강점: 도발·반격 생존 — 방어력, 보호막, 도발
- 약점: 낮은 화력 — 딜 기여도 낮음

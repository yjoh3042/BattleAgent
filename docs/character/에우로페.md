# 에우로페 — 스킬 컨셉 상세 기획서

## 캐릭터 아이덴티티
| 항목 | 값 |
|---|---|
| 속성 | Forest |
| 직업 | Attacker |
| 성급 | 1★ |

## 스킬 상세

### 노멀
- 설명: 적 1명 대미지
- 강화+1: 대미지 +10%
- 강화+2: 30% 확률로 대상 방어력 감소

### 액티브
- 설명: 적 1명 대미지 + 넉백 (전열→후열)
- 강화+1: 대미지 +15%
- 강화+2: 넉백된 적 다음 턴 속도 -20

### 얼티밋
- 설명: 적 전열 대미지 + 전열의 적 후열로 넉백
- 강화+1: 대미지 +20%
- 강화+2: 넉백 후 빈 전열에 아군 풀 (위치 교환)

### 패시브
- 설명: 후열의 적 공격 시 추가 대미지
- 강화+1: 후열 추가 대미지 수치 +15%
- 강화+2: 넉백 성공 시 자신 속도 +10 2턴

## 속성 상성
- 유리: 수속성
- 불리: 화염속성

## 핵심 포인트
- 강점: 고배율 공격 — 기본기 충실
- 약점: 낮은 방어력 — 집중 공격에 취약

# 샤를 — 스킬 컨셉 상세 기획서

## 캐릭터 아이덴티티
| 항목 | 값 |
|---|---|
| 속성 | Light |
| 직업 | Defender |
| 성급 | 1★ |

## 스킬 상세

### 노멀
- 설명: 적 1명 2타 대미지
- 강화+1: 피격 시 자힐 5%
- 강화+2: 적중 시 자신에게 소량 보호막

### 액티브
- 설명: 자신 방어력 증가
- 강화+1: 버프 수치 +10% 증가
- 강화+2: 사용 후 인접 아군에게 보호막 부여

### 얼티밋
- 설명: 직선의 적 대미지
- 강화+1: 대미지 +20%
- 강화+2: 보호막이 깨지지 않은 아군 공격력 +15% 2턴

### 패시브
- 설명: 체력이 50%이하일 경우 방어력 증가
- 강화+1: 방어력 증가 수치 +15%
- 강화+2: 보호막 파괴 시 자신 방어력 +15% 2턴

## 속성 상성
- 유리: 암속성
- 불리: 없음속성

## 핵심 포인트
- 강점: 도발·반격 생존 — 방어력
- 약점: 낮은 화력 — 딜 기여도 낮음

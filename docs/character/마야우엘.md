# 마야우엘 — 스킬 컨셉 상세 기획서

## 캐릭터 아이덴티티
| 항목 | 값 |
|---|---|
| 속성 | Water |
| 직업 | Magician |
| 성급 | 1★ |

## 스킬 상세

### 노멀
- 설명: 적 1명 대미지
- 강화+1: 대미지 +10%
- 강화+2: 30% 확률로 속도 감소

### 액티브
- 설명: 적 전체 대미지
- 강화+1: 대미지 +15%
- 강화+2: 적중 시 50% 확률로 빙결 1턴

### 얼티밋
- 설명: 적 전체 대미지 + 30% 확률로 빙결
- 강화+1: 대미지 +20%, 빙결 확률 +20%
- 강화+2: 속도가 감소된 적에게 추가 대미지 +30%

### 패시브
- 설명: 빙결 중인 적에게 대미지 증가
- 강화+1: 추가 대미지 수치 +15%
- 강화+2: 빙결 부여 시 대상 속도 추가 감소

## 속성 상성
- 유리: 화염속성
- 불리: 목속성

## 핵심 포인트
- 강점: 광역 CC/디버프 — 빙결, 전체 대미지
- 약점: 낮은 HP/DEF — 선공을 잡히면 위험

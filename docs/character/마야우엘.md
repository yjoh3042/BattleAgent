## 3. 설계 의도

**기본 역할 설계**: 빠른 SPD(100)로 선제 행동하여 CC와 디버프로 적을 제압. 딜보다는 적 행동 방해와 전장 통제에 특화된 컨트롤러.

**고유 메커니즘**:

- **SP 잠금**: 적의 SP 충전을 일정 턴 차단. SP 강탈과 함께 적 얼티밋을 완전히 봉쇄하는 콤보 가능.

**트리거 설계 의도**:

- **ON_HIT**: 피격 시 패시브 발동 (1회). 도발/탱킹과 연동하여 맞으면 되받아치는 방어적 공격 패턴.

## 4. 이상적 전투 흐름

> CTB 행동 간격: **2.50초** (SPD 120)
> SP 요구량: **4** (팀 공유)
> 액티브 쿨타임: **3턴**

**[T1]** (행동시간 2.50s — 서포터 다음으로 빠름) 「물의 채찍」 사용
  → 선제 공격 + CC/디버프 부여. 적 딜러보다 먼저 행동하여 위협 제거.

**[T2]** 「물의 채찍」 반복
  → CC 재적용 또는 디버프 유지. 적의 행동 기회를 계속 줄임.

**[T3]** 「빙결의 손길」 발동 (쿨타임 해제)
  → 핵심 CC/디버프 콤보. 적 핵심 유닛을 무력화.

**[T4]** (SP 4 충족 시) 「얼어붙은 세계」 발동 → **엑스트라 턴**
  → AoE CC/디버프로 적 전체 제압. 엑스트라 턴에서 노말로 추가 CC.

# 아슈토레스 — 스킬 컨셉 상세 기획서

## 캐릭터 아이덴티티
| 항목 | 값 |
|---|---|
| 속성 | Light |
| 직업 | Attacker |
| 성급 | 3★ |

## 스킬 상세

### 노멀
- 설명: 적 1명 대미지
- 강화+1: 대미지 +10%
- 강화+2: 30% 확률로 대상 버프 1개 제거

### 액티브
- 설명: 적 전열 우선 적에게 대미지
- 강화+1: 대미지 +15%
- 강화+2: 보호막이 있는 적 대상 보호막 무시 대미지

### 얼티밋
- 설명: 적 전열 우선 적에게 대미지 + 디버프 폭발(디버프를 제거하면서 디버프x대미지)
- 강화+1: 제거 개수 +1 추가
- 강화+2: 버프가 없는 적에게 추가 대미지 +30%

### 패시브
- 설명: 디버프 중인 적에게 추가대미지
- 강화+1: 추가 대미지 수치 +15%
- 강화+2: 버프 제거 성공 시 자신 공격력 +10% 2턴

## 속성 상성
- 유리: 암속성
- 불리: 없음속성

## 핵심 포인트
- 강점: 고배율 공격 — 기본기 충실
- 약점: 낮은 방어력 — 집중 공격에 취약

# 메티스 — 스킬 컨셉 상세 기획서

## 캐릭터 아이덴티티
| 항목 | 값 |
|---|---|
| 속성 | Forest |
| 직업 | Defender |
| 성급 | 3★ |

## 스킬 상세

### 노멀
- 설명: 적 1명 대미지
- 강화+1: 피격 시 자힐 5%
- 강화+2: 적중 시 자신 방어력 소폭 증가

### 액티브
- 설명: 자신 방어력 증가 + 가장 가까운 아군에게 보호막 부여
- 강화+1: 버프 수치 +10% 증가
- 강화+2: 사용 후 2턴간 받는 대미지 -15%

### 얼티밋
- 설명: 적 전체 대미지 + 아군 전체 방어력 증가
- 강화+1: 대미지 +20%
- 강화+2: 사용 후 3턴간 아군 전체 방어력 +15%

### 패시브
- 설명: 버프가 2개 이상인 아군이 받는 대미지 감소
- 강화+1: 추가 대미지 수치 +15%
- 강화+2: 버프 1개당 받는 대미지 추가 -3%

## 속성 상성
- 유리: 수속성
- 불리: 화염속성

## 핵심 포인트
- 강점: 도발·반격 생존 — 방어력, 보호막, 전체 대미지
- 약점: 낮은 화력 — 딜 기여도 낮음

## 4. 이상적 전투 흐름

> CTB 행동 간격: **3.00초** (SPD 100)
> SP 요구량: **3** (팀 공유)
> 액티브 쿨타임: **3턴**

**[전투 시작]** 「달빛 보호막」 자동 발동
  → `보호막` maxHP×8% → 아군 전체 / `최대HP 비율 회복` 100% (heal_ratio=0.1) → 아군 전체

**[T1]** (행동시간 3.00s — SPD 110으로 딜러보다 먼저 행동) 「이클립스」 사용
  → 기본 공격으로 어그로 생성. 전열 배치로 적 공격을 자연스럽게 유도.

**[T2]** 「트릭스타」 발동 (핵심)
  → 도발/보호막/반격 준비 등 방어 체계 구축. 이 시점부터 팀 방어막 역할 시작.

**[T3~4]** 「이클립스」 반복 + 피격 유도
  → 도발 상태에서 적 공격을 받으며 반격/패시브 발동으로 딜 기여.

**[T4~5]** (SP 3 충족 시) 「사랑을 담아」 발동 → **엑스트라 턴**
  → 팀 보호 얼티밋으로 결정적 순간에 팀을 지킴.

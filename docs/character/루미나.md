# 루미나 — 스킬 컨셉 상세 기획서

## 캐릭터 아이덴티티
| 항목 | 값 |
|---|---|
| 속성 | Light |
| 직업 | Magician |
| 성급 | 3.5★ |

## 스킬 상세

### 노멀
- 설명: 적 1명 대미지
- 강화+1: 대미지 +10%
- 강화+2: 30% 확률로 대상 버프 1개 제거

### 액티브
- 설명: 적 1명 대미지 + 아군 전체 디버프 1개 제거
- 강화+1: 대미지 +15%
- 강화+2: 적의 버프 1개당 추가 대미지 +5%

### 얼티밋
- 설명: 적 전체 대미지 + 디버프 중인 아군에게 보호막 부여
- 강화+1: 보호막 수치 +20%
- 강화+2: 디버프 중인 적 수만큼 아군 보호막 강화

### 패시브
- 설명: 아군의 디버프가 제거될 때마다 제거된 아군 공격력 증가
- 강화+1: 추가 대미지 수치 +15%
- 강화+2: 디버프 제거 시 제거한 수만큼 자신 공격력 증가

## 속성 상성
- 유리: 암속성
- 불리: 없음속성

## 핵심 포인트
- 강점: 광역 CC/디버프 — 공격력, 보호막, 전체 대미지
- 약점: 낮은 HP/DEF — 선공을 잡히면 위험

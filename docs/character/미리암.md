# 미리암 — 스킬 컨셉 상세 기획서

## 캐릭터 아이덴티티
| 항목 | 값 |
|---|---|
| 속성 | Forest |
| 직업 | Attacker |
| 성급 | 3★ |

## 스킬 상세

### 노멀
- 설명: 적 1명 대미지
- 강화+1: 대미지 +10%
- 강화+2: 30% 확률로 대상 방어력 감소

### 액티브
- 설명: 자신 HP 10% 소모 + 적 1명 대미지 + 중독 부여
- 강화+1: HP 소모량 10% → 8%로 감소
- 강화+2: 중독 2스택 이상인 적에게 방어력 무시

### 얼티밋
- 설명: 적 전체 대미지 + 중독 중인 적 추가 대미지
- 강화+1: 대미지 +20%, 중독 스택 +1 추가
- 강화+2: 중독 중인 적 수만큼 자신 HP 회복

### 패시브
- 설명: 중독 중인 적이 많을수록 자신 공격력 증가
- 강화+1: 중독 1명당 공격력 증가 수치 +5%
- 강화+2: 중독 적이 3명 이상이면 자신 치명타 확률 +20%

## 속성 상성
- 유리: 수속성
- 불리: 화염속성

## 핵심 포인트
- 강점: 고배율 공격 — 중독, 공격력, 전체 대미지
- 약점: 낮은 방어력 — 집중 공격에 취약

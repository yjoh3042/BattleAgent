## 4. 이상적 전투 흐름

> CTB 행동 간격: **3.75초** (SPD 80)
> SP 요구량: **6** (팀 공유)
> 액티브 쿨타임: **3턴**

**[전투 시작]** 「분노의 씨앗」 자동 발동
  → `스탯 변경` atk ↑20%% (2턴) → 자기 자신

**[T1]** (행동시간 3.75s) 「웨이팅 포 딜」 사용
  → 가장 가까운 적에게 기본 딜. 서포터/구속형이 먼저 움직여 디버프를 걸어둔 상태가 이상적.

**[T2]** (행동시간 7.50s) 「웨이팅 포 딜」 사용
  → 노말 반복으로 딜 축적. 액티브 쿨타임 대기.

**[T3]** (행동시간 11.25s) 「크로스 더 루비콘」 사용 (액티브 쿨 해제)
  → 핵심 액티브 스킬 발동. 추가 효과(화상/방깎/CC 등)로 적 제압력 강화.

**[T4~5]** (SP 6 충족 시) 「퓨리오스」 발동 → **엑스트라 턴**
  → 얼티밋으로 최대 딜. 엑스트라 턴에서 노말/액티브 추가 사용.

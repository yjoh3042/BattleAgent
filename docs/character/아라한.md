# 아라한 — 스킬 컨셉 상세 기획서

## 캐릭터 아이덴티티
| 항목 | 값 |
|---|---|
| 속성 | Water |
| 직업 | Supporter |
| 성급 | 3.5★ |

## 스킬 상세

### 노멀
- 설명: 적 1명 대미지
- 강화+1: 대미지 +10%
- 강화+2: 적중 시 아군 1명 속도 소폭 증가

### 액티브
- 설명: 적 1명 대미지 + 자신의 속도가 150 이상일 경우 (SP+1)
- 강화+1: SP 획득 조건 완화
- 강화+2: 수속성 아군 추가로 속도 +10

### 얼티밋
- 설명: 적 전체 대미지 + 아군 전체 속도 증가
- 강화+1: 속도 증가 수치 +10 추가
- 강화+2: 사용 후 가장 빠른 아군 즉시 턴 획득

### 패시브
- 설명: 아군이 대미지를 줄 때 속도에 비례해서 대미지 증가 (속도 200 상한)
- 강화+1: 속도 상한 200 → 220
- 강화+2: 속도 180 이상인 아군 치명타 확률 +10%

## 속성 상성
- 유리: 화염속성
- 불리: 목속성

## 핵심 포인트
- 강점: 팀 버프·시너지 — 속도, 전체 대미지
- 약점: 낮은 단독 전투력 — 아군 없이 무력

# 데이노 — 스킬 컨셉 상세 기획서

## 캐릭터 아이덴티티
| 항목 | 값 |
|---|---|
| 속성 | Water |
| 직업 | Defender |
| 성급 | 1★ |

## 스킬 상세

### 노멀
- 설명: 적 1명 대미지
- 강화+1: 피격 시 자힐 5%
- 강화+2: 적중 시 자신 속도 소폭 증가

### 액티브
- 설명: 적 1명 대미지 + 자신 방어력 증가
- 강화+1: 버프 수치 +10% 증가
- 강화+2: 사용 후 2턴간 자신 회피율 증가

### 얼티밋
- 설명: 전열의 적 대미지 + 도발
- 강화+1: 도발 지속 +1턴
- 강화+2: 사용 후 아군 전체 속도 +10 증가 2턴

### 패시브
- 설명: 피격 시 자신 속도 소폭 증가
- 강화+1: 속도 관련 수치 +10
- 강화+2: 속도가 높을수록 받는 대미지 추가 감소

## 속성 상성
- 유리: 화염속성
- 불리: 목속성

## 핵심 포인트
- 강점: 도발·반격 생존 — 속도, 방어력, 도발
- 약점: 낮은 화력 — 딜 기여도 낮음

# 티와즈 — 스킬 컨셉 상세 기획서

## 캐릭터 아이덴티티
| 항목 | 값 |
|---|---|
| 속성 | Light |
| 직업 | Healer |
| 성급 | 3★ |

## 스킬 상세

### 노멀
- 설명: 적 1명 대미지
- 강화+1: 대미지 +10%
- 강화+2: 적중 시 아군 최저HP 소량 회복

### 액티브
- 설명: 사망한 아군 1명 HP 30%로 부활
- 강화+1: 부활 HP +15% 증가
- 강화+2: 부활 대상에게 2턴간 받는 대미지 -20%

### 얼티밋
- 설명: 아군 한명 부활
- 강화+1: 부활 HP +20% 증가
- 강화+2: 부활 대상에게 3턴간 받는 대미지 -30%

### 패시브
- 설명: 아군이 부활할 때 부활 대상에게 보호막 부여
- 강화+1: 보호막 수치 +15%
- 강화+2: 부활 아군에게 추가로 디버프 면역 2턴

## 속성 상성
- 유리: 암속성
- 불리: 없음속성

## 핵심 포인트
- 강점: 팀 회복·정화 — 보호막, 부활
- 약점: 낮은 공격력 — 전투 기여는 회복에 의존

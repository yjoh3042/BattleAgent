# 메두사 — 스킬 컨셉 상세 기획서

## 캐릭터 아이덴티티
| 항목 | 값 |
|---|---|
| 속성 | Dark |
| 직업 | Attacker |
| 성급 | 1★ |

## 스킬 상세

### 노멀
- 설명: 적 1명 대미지
- 강화+1: 대미지 +10%
- 강화+2: 치명타 시 추가 대미지 +15%

### 액티브
- 설명: 적 1명 대미지 + 50% 확률로 석화 (1턴 행동불가)
- 강화+1: 석화 확률 50% → 65%
- 강화+2: 석화 중인 적에게 방어력 무시 대미지

### 얼티밋
- 설명: 적 전체 대미지 + 30% 확률로 석화
- 강화+1: 대미지 +20%, 석화 확률 +15%
- 강화+2: 석화 적 수만큼 자신 공격력 +10% 증가

### 패시브
- 설명: 석화 중인 적 공격 시 대미지 2배
- 강화+1: 석화 대미지 배율 2배 → 2.5배
- 강화+2: 석화 해제 시 대상에게 방어력 감소 2턴

## 속성 상성
- 유리: 광속성
- 불리: 없음속성

## 핵심 포인트
- 강점: 고배율 공격 — 석화, 전체 대미지
- 약점: 낮은 방어력 — 집중 공격에 취약

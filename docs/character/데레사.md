## 3. 설계 의도

**기본 역할 설계**: 높은 DEF/HP와 도발로 아군을 보호하는 탱커. 적의 공격을 집중시키고 반격/CC로 방어적 가치를 제공.

**트리거 설계 의도**:

- **ON_HIT**: 피격 시 패시브 발동 (1회). 도발/탱킹과 연동하여 맞으면 되받아치는 방어적 공격 패턴.

### 밸런스 변경 이력

- **v7.3**: 버프: 5.00→6.00
- **v7.3**: 버프: 2→3턴

## 4. 이상적 전투 흐름

> CTB 행동 간격: **2.73초** (SPD 110)
> SP 요구량: **3** (팀 공유)
> 액티브 쿨타임: **3턴**

**[T1]** (행동시간 2.73s — SPD 110으로 딜러보다 먼저 행동) 「스틸 웨일링」 사용
  → 기본 공격으로 어그로 생성. 전열 배치로 적 공격을 자연스럽게 유도.

**[T2]** 「데털민트 소드」 발동 (핵심)
  → 도발/보호막/반격 준비 등 방어 체계 구축. 이 시점부터 팀 방어막 역할 시작.

**[T3~4]** 「스틸 웨일링」 반복 + 피격 유도
  → 도발 상태에서 적 공격을 받으며 반격/패시브 발동으로 딜 기여.
  → 피격 시 「불꽃 반격」 발동 — 반격 + 추가 효과

**[T4~5]** (SP 3 충족 시) 「새크리파이스」 발동 → **엑스트라 턴**
  → 팀 보호 얼티밋으로 결정적 순간에 팀을 지킴.

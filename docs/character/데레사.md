# 데레사 — 스킬 컨셉 상세 기획서

## 캐릭터 아이덴티티
| 항목 | 값 |
|---|---|
| 속성 | Fire |
| 직업 | Defender |
| 성급 | 3★ |

## 스킬 상세

### 노멀
- 설명: 적 1명 대미지
- 강화+1: 피격 시 자힐 5%
- 강화+2: 30% 확률로 공격자에게 화상

### 액티브
- 설명: 적 1명 대미지 및 자신 반격확률 증가
- 강화+1: 반격 확률 +15%
- 강화+2: 사용 후 2턴간 피격 시 공격자에게 화상 부여

### 얼티밋
- 설명: 적 전열 대미지 및 50% 확률로 기절
- 강화+1: 대미지 +20%, 기절 확률 +20%
- 강화+2: 사용 후 3턴간 아군 전체 받는 대미지 -10%

### 패시브
- 설명: 반격 시 대상에게 화상 부여
- 강화+1: 반격 대미지 +20%
- 강화+2: 피격 시 20% 확률로 자힐 10%

## 속성 상성
- 유리: 목속성
- 불리: 수속성

## 핵심 포인트
- 강점: 도발·반격 생존 — 화상, 기절, 반격
- 약점: 낮은 화력 — 딜 기여도 낮음

# 도계화 — 스킬 컨셉 상세 기획서

## 캐릭터 아이덴티티
| 항목 | 값 |
|---|---|
| 속성 | Water |
| 직업 | Magician |
| 성급 | 3★ |

## 스킬 상세

### 노멀
- 설명: 적 1명 대미지
- 강화+1: 대미지 +10%
- 강화+2: 30% 확률로 속도 감소

### 액티브
- 설명: 전 1명 대미지 + 속도 감소
- 강화+1: 대미지 +15%
- 강화+2: 적중 시 50% 확률로 빙결 1턴

### 얼티밋
- 설명: 적 후열 우선 대미지 대상 대미지 + 빙결
- 강화+1: 대미지 +20%, 빙결 확률 +20%
- 강화+2: 속도가 감소된 적에게 추가 대미지 +30%

### 패시브
- 설명: 거리가 먼 적 공격 시 30% 확률로 추가 타격
- 강화+1: 추가 타격 확률 30% → 45%
- 강화+2: 추가 타격 시 속도 감소 부여

## 속성 상성
- 유리: 화염속성
- 불리: 목속성

## 핵심 포인트
- 강점: 광역 CC/디버프 — 빙결, 속도
- 약점: 낮은 HP/DEF — 선공을 잡히면 위험

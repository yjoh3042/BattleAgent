## 4. 이상적 전투 흐름

> CTB 행동 간격: **3.33초** (SPD 90)
> SP 요구량: **3** (팀 공유)
> 액티브 쿨타임: **3턴**

**[전투 시작]** 「바람의 가호」 자동 발동
  → `최대HP 비율 회복` 100% (heal_ratio=0.15) → 아군 전체 / `보호막` maxHP×6% → 아군 전체

**[T1]** (행동시간 3.33s) 「바람의 속삭임」 사용
  → 아직 아군이 건강한 상태. 기본 공격으로 딜 기여.

**[T2~3]** 「회복의 바람」 발동 (핵심 회복)
  → 아군 피해가 누적되기 시작. HP가 가장 낮은 아군을 우선 회복.

**[T3~4]** (SP 3 충족 — 힐러는 SP 3으로 빠른 얼티밋) 「대기의 은혜」 발동 → **엑스트라 턴**
  → 전체 회복 + 추가 효과. 팀 HP를 한번에 끌어올리는 결정적 힐.
  → 엑스트라 턴에서 추가 노말/액티브로 이중 힐 가능.

**[T5]** 「회복의 바람」 재발동
  → 액티브 쿨타임 돌아옴. 팀 HP 유지 관리 지속.

# 에우로스 — 스킬 컨셉 상세 기획서

## 캐릭터 아이덴티티
| 항목 | 값 |
|---|---|
| 속성 | Water |
| 직업 | Healer |
| 성급 | 2★ |

## 스킬 상세

### 노멀
- 설명: 적 1명 대미지
- 강화+1: 대미지 +10%
- 강화+2: 적중 시 아군 최저HP 소량 회복

### 액티브
- 설명: 체력이 낮은 아군 1명 회복 + 속도 증가 버프
- 강화+1: 회복량 +15%
- 강화+2: 회복 대상의 속도 +15 증가 2턴

### 얼티밋
- 설명: 아군 전체 회복 + 속도 증가
- 강화+1: 속도 증가 수치 +10 추가
- 강화+2: 회복 후 아군 전체 회피율 증가 2턴

### 패시브
- 설명: 턴 시작 시 속도가 가장 높은 아군 소량 회복
- 강화+1: 회복량 +15%
- 강화+2: 회복 시 대상 디버프 1개 추가 제거

## 속성 상성
- 유리: 화염속성
- 불리: 목속성

## 핵심 포인트
- 강점: 팀 회복·정화 — 속도
- 약점: 낮은 공격력 — 전투 기여는 회복에 의존

# 자청비 — 스킬 컨셉 상세 기획서

## 캐릭터 아이덴티티
| 항목 | 값 |
|---|---|
| 속성 | Forest |
| 직업 | Supporter |
| 성급 | 2★ |

## 스킬 상세

### 노멀
- 설명: 적 1명 대미지
- 강화+1: 대미지 +10%
- 강화+2: 적중 시 아군 1명 방어력 소폭 증가

### 액티브
- 설명: 가장 가까운 아군 방어력 증가 + 지속 회복 부여
- 강화+1: 회복량 +15%
- 강화+2: 목속성 아군 추가로 방어력 +10%

### 얼티밋
- 설명: 아군 전체 방어력 증가 + 디버프 1개 제거
- 강화+1: 제거 개수 +1 추가
- 강화+2: 목속성 아군 3턴간 방어력 +20% 추가 버프

### 패시브
- 설명: 목속성 아군이 피격 시 받는 대미지 소폭 감소
- 강화+1: 추가 대미지 수치 +15%
- 강화+2: 목속성 아군 디버프 지속 -1턴 감소

## 속성 상성
- 유리: 수속성
- 불리: 화염속성

## 핵심 포인트
- 강점: 팀 버프·시너지 — 방어력
- 약점: 낮은 단독 전투력 — 아군 없이 무력

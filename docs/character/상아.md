# 상아 — 스킬 컨셉 상세 기획서

## 캐릭터 아이덴티티
| 항목 | 값 |
|---|---|
| 속성 | Water |
| 직업 | Supporter |
| 성급 | 3★ |

## 스킬 상세

### 노멀
- 설명: 적 1명 대미지
- 강화+1: 대미지 +10%
- 강화+2: 적중 시 아군 1명 속도 소폭 증가

### 액티브
- 설명: 적 1명 대미지 + 속도 감소, 속도 감소 중인 적에게 30% 빙결
- 강화+1: 속도 감소 수치 +10
- 강화+2: 빙결 확률 30% → 50%

### 얼티밋
- 설명: 수속성 아군 공격력 증가
- 강화+1: 대미지 +20%
- 강화+2: 사용 후 가장 빠른 아군 즉시 턴 획득

### 패시브
- 설명: 빙결 중인 적이 있을 때 수속성 아군 속도 추가 +10
- 강화+1: 속도 추가 증가 +10 → +15
- 강화+2: 빙결 적이 2명 이상이면 수속성 아군 치명타 +10%

## 속성 상성
- 유리: 화염속성
- 불리: 목속성

## 핵심 포인트
- 강점: 팀 버프·시너지 — 빙결, 속도, 공격력
- 약점: 낮은 단독 전투력 — 아군 없이 무력

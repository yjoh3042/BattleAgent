## 3. 설계 의도

**기본 역할 설계**: 가장 빠른 SPD(120)으로 매 라운드 먼저 행동. 버프/디버프로 팀 전투력을 극대화하는 인에이블러.

**고유 메커니즘**:

- **SP 강탈**: 적 팀의 SP를 빼앗아 얼티밋 사용을 지연시키는 자원 전쟁 설계. 적의 폭딜 타이밍을 무너뜨리는 핵심 견제.
- **SP 잠금**: 적의 SP 충전을 일정 턴 차단. SP 강탈과 함께 적 얼티밋을 완전히 봉쇄하는 콤보 가능.

**트리거 설계 의도**:

- **ON_HIT**: 피격 시 패시브 발동 (1회). 도발/탱킹과 연동하여 맞으면 되받아치는 방어적 공격 패턴.

## 4. 이상적 전투 흐름

> CTB 행동 간격: **2.50초** (SPD 120)
> SP 요구량: **4** (팀 공유)
> 액티브 쿨타임: **3턴**

**[T1]** (행동시간 2.50s — **팀 내 가장 빠른 행동**) 「수류」 사용
  → 모든 아군보다 먼저 행동. 기본 공격으로 전장 정리 시작.

**[T2]** 「수렁」 발동 (핵심 버프/디버프)
  → 팀 전체에 버프 부여 또는 적에게 디버프. 이후 아군 딜러의 공격이 강화된 상태로 타격.

**[T3]** 「수류」 반복
  → 빠른 SPD로 다시 행동. 적 위협 제거 또는 추가 지원.

**[T4]** (SP 4 충족 시) 「대조수」 발동 → **엑스트라 턴**
  → 팀 전체 강화 또는 적 전체 약화. 엑스트라 턴에서 액티브 재사용으로 이중 버프.

**[T5]** 버프 갱신 사이클
  → 2턴짜리 버프가 만료되는 시점. 액티브 재발동으로 버프 유지.

## 3. 설계 의도

**기본 역할 설계**: 빠른 SPD(100)로 선제 행동하여 CC와 디버프로 적을 제압. 딜보다는 적 행동 방해와 전장 통제에 특화된 컨트롤러.

**트리거 설계 의도**:

- **ON_BATTLE_START**: 전투 시작 즉시 패시브 자동 발동. 사전 버프/힐/DoT로 라운드 1부터 이점을 확보하는 선제 설계.

## 4. 이상적 전투 흐름

> CTB 행동 간격: **3.00초** (SPD 100)
> SP 요구량: **4** (팀 공유)
> 액티브 쿨타임: **3턴**

**[전투 시작]** 「화염 증폭」 자동 발동
  → `화상` maxHP×10% (2턴, 최대 3스택) → 가장 가까운 적 1체

**[T1]** (행동시간 3.00s — 서포터 다음으로 빠름) 「다비의 불꽃」 사용
  → 선제 공격 + CC/디버프 부여. 적 딜러보다 먼저 행동하여 위협 제거.

**[T2]** 「다비의 불꽃」 반복
  → CC 재적용 또는 디버프 유지. 적의 행동 기회를 계속 줄임.

**[T3]** 「화상 촉진」 발동 (쿨타임 해제)
  → 핵심 CC/디버프 콤보. 적 핵심 유닛을 무력화.

**[T4]** (SP 4 충족 시) 「업화」 발동 → **엑스트라 턴**
  → AoE CC/디버프로 적 전체 제압. 엑스트라 턴에서 노말로 추가 CC.

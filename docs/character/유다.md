## 3. 설계 의도

**기본 역할 설계**: 높은 DEF/HP와 도발로 아군을 보호하는 탱커. 적의 공격을 집중시키고 반격/CC로 방어적 가치를 제공.

**트리거 설계 의도**:

- **ON_BATTLE_START**: 전투 시작 즉시 패시브 자동 발동. 사전 버프/힐/DoT로 라운드 1부터 이점을 확보하는 선제 설계.

### 밸런스 변경 이력

출시 이후 밸런스 변경 없음.

## 4. 이상적 전투 흐름

> CTB 행동 간격: **2.73초** (SPD 110)
> SP 요구량: **3** (팀 공유)
> 액티브 쿨타임: **3턴**

**[전투 시작]** 「방화벽」 자동 발동
  → `보호막` maxHP×10% → 자기 자신

**[T1]** (행동시간 2.73s — SPD 110으로 딜러보다 먼저 행동) 「마탄」 사용
  → 기본 공격으로 어그로 생성. 전열 배치로 적 공격을 자연스럽게 유도.

**[T2]** 「악마의 방패」 발동 (핵심)
  → 도발/보호막/반격 준비 등 방어 체계 구축. 이 시점부터 팀 방어막 역할 시작.

**[T3~4]** 「마탄」 반복 + 피격 유도
  → 도발 상태에서 적 공격을 받으며 반격/패시브 발동으로 딜 기여.

**[T4~5]** (SP 3 충족 시) 「심판」 발동 → **엑스트라 턴**
  → 팀 보호 얼티밋으로 결정적 순간에 팀을 지킴.

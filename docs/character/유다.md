# 유다 — 스킬 컨셉 상세 기획서

| 항목 | 값 |
|---|---|
| 속성 | Fire |
| 직업 | Defender |

## 스킬 상세

### 노멀
- 설명: 적 1명 대미지
- 강화+1: 피격 시 자힐 5%
- 강화+2: 30% 확률로 공격자에게 화상

### 액티브
- 설명: 적 1명 대미지 + 자신 방어력 증가
- 강화+1: 버프 수치 +10% 증가
- 강화+2: 사용 후 2턴간 피격 시 공격자에게 화상 부여

### 얼티밋
- 설명: 전열의 적 대미지 + 도발
- 강화+1: 도발 지속 +1턴
- 강화+2: 사용 후 3턴간 아군 전체 받는 대미지 -10%

### 패시브
- 설명: 피격 시 30% 확률로 공격자에게 화상 부여
- 강화+1: 발동 확률 30% → 45%
- 강화+2: 피격 시 20% 확률로 자힐 10%

## 속성 상성
- 유리: 목속성
- 불리: 수속성

## 핵심 포인트
- 강점: 도발·반격 생존 — 화상, 방어력, 도발
- 약점: 낮은 화력 — 딜 기여도 낮음

# 반시 — 스킬 컨셉 상세 기획서

## 캐릭터 아이덴티티
| 항목 | 값 |
|---|---|
| 속성 | Dark |
| 직업 | Magician |
| 성급 | 3★ |

## 스킬 상세

### 노멀
- 설명: 적 1명 대미지
- 강화+1: 대미지 +10%
- 강화+2: 치명타 시 추가 대미지 +15%

### 액티브
- 설명: 자신과 무작위 적 1명에게 적중률 증가 버프부여
- 강화+1: 버프 수치 +10% 증가
- 강화+2: 치명타 확률 중첩 최대치 +1 증가

### 얼티밋
- 설명: 적 전체 대미지
- 강화+1: 대미지 +20%
- 강화+2: 침묵 또는 출혈 중인 적에게 추가 대미지 +30%

### 패시브
- 설명: 버프가 3개 이상인 적 공격시 50% 확률로 기절 부여
- 강화+1: 발동 확률 50% → 65%
- 강화+2: 침묵 부여 시 대상 받는 대미지 +10% 2턴

## 속성 상성
- 유리: 광속성
- 불리: 없음속성

## 핵심 포인트
- 강점: 광역 CC/디버프 — 기절, 전체 대미지
- 약점: 낮은 HP/DEF — 선공을 잡히면 위험

# 다비2 — 스킬 컨셉 상세 기획서

## 캐릭터 아이덴티티
| 항목 | 값 |
|---|---|
| 속성 | Fire |
| 직업 | Magician |
| 성급 | 3★ |

## 스킬 상세

### 노멀
- 설명: 적 1명 대미지
- 강화+1: 대미지 +10%
- 강화+2: 30% 확률로 화상 부여

### 액티브
- 설명: 적 1명 대미지 + 화상 부여, 대상 주변 적에게 화상 전이
- 강화+1: 전이 범위 +1명 추가
- 강화+2: 화상 중인 적에게 치명타 확률 +30%

### 얼티밋
- 설명: 적 전체 대미지 + 화상 중인 적 추가 대미지
- 강화+1: 대미지 +20%, 화상 스택 +1 추가
- 강화+2: 화상 중인 적 수만큼 전체 대미지 +10% 증가

### 패시브
- 설명: 화상 중인 적이 2명 이상일 때 자신 공격력 증가
- 강화+1: 추가 대미지 수치 +15%
- 강화+2: 화상 부여 시 30% 확률로 화상 스택 +1 추가

## 속성 상성
- 유리: 목속성
- 불리: 수속성

## 핵심 포인트
- 강점: 광역 CC/디버프 — 화상, 공격력, 전체 대미지
- 약점: 낮은 HP/DEF — 선공을 잡히면 위험

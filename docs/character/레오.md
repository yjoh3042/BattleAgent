# 레오 — 스킬 컨셉 상세 기획서

## 캐릭터 아이덴티티
| 항목 | 값 |
|---|---|
| 속성 | Water |
| 직업 | Supporter |
| 성급 | 1★ |

## 스킬 상세

### 노멀
- 설명: 적 1명 대미지
- 강화+1: 대미지 +10%
- 강화+2: 적중 시 아군 1명 속도 소폭 증가

### 액티브
- 설명: 적 1명 대미지 + 아군 1명 회피율 증가
- 강화+1: 회피율 증가 수치 +10%
- 강화+2: 회피율 버프 대상에게 추가로 속도 +10

### 얼티밋
- 설명: 아군 전체 속도 증가
- 강화+1: 속도 증가 수치 +10 추가
- 강화+2: 사용 후 가장 빠른 아군 즉시 턴 획득

### 패시브
- 설명: 아군이 공격을 회피할 때마다 회피한 아군 속도 +5
- 강화+1: 회피 시 속도 증가 +5 → +8
- 강화+2: 회피 3회 누적 시 해당 아군 추가턴 획득

## 속성 상성
- 유리: 화염속성
- 불리: 목속성

## 핵심 포인트
- 강점: 팀 버프·시너지 — 속도
- 약점: 낮은 단독 전투력 — 아군 없이 무력

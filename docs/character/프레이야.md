## 3. 설계 의도

**기본 역할 설계**: 아군 HP 회복과 디버프 정화로 팀 생존을 보장. SP Cost 3으로 빠른 얼티밋 회전이 가능하여 지속적 지원 가능.

**트리거 설계 의도**:

- **ON_BATTLE_START**: 전투 시작 즉시 패시브 자동 발동. 사전 버프/힐/DoT로 라운드 1부터 이점을 확보하는 선제 설계.

### 밸런스 변경 이력

## 4. 이상적 전투 흐름

> CTB 행동 간격: **3.33초** (SPD 90)
> SP 요구량: **3** (팀 공유)
> 액티브 쿨타임: **3턴**

**[전투 시작]** 「자연의 치유」 자동 발동
  → `최대HP 비율 회복` 100% (heal_ratio=0.15) → HP가 가장 낮은 아군

**[T1]** (행동시간 3.33s) 「자연의 가시」 사용
  → 아직 아군이 건강한 상태. 기본 공격으로 딜 기여.

**[T2~3]** 「생명의 꽃」 발동 (핵심 회복)
  → 아군 피해가 누적되기 시작. HP가 가장 낮은 아군을 우선 회복.

**[T3~4]** (SP 3 충족 — 힐러는 SP 3으로 빠른 얼티밋) 「대지의 은총」 발동 → **엑스트라 턴**
  → 전체 회복 + 추가 효과. 팀 HP를 한번에 끌어올리는 결정적 힐.
  → 엑스트라 턴에서 추가 노말/액티브로 이중 힐 가능.

**[T5]** 「생명의 꽃」 재발동
  → 액티브 쿨타임 돌아옴. 팀 HP 유지 관리 지속.

## 3. 설계 의도

**기본 역할 설계**: 가장 빠른 SPD(120)으로 매 라운드 먼저 행동. 버프/디버프로 팀 전투력을 극대화하는 인에이블러.

**트리거 설계 의도**:

- **ON_BATTLE_START**: 전투 시작 즉시 패시브 자동 발동. 사전 버프/힐/DoT로 라운드 1부터 이점을 확보하는 선제 설계.
- **ON_CRITICAL_HIT**: 크리티컬 적중 시 패시브 발동. CRI 버프와 연동하면 매 공격마다 추가 액션이 발생하는 연쇄 설계.

### 밸런스 변경 이력

- **v8**: ALL_ALLY→ALLY_ADJACENT (CRI buffs
- **v8**: ON_CRITICAL_HIT 추가 (크리 시 자힐

## 4. 이상적 전투 흐름

> CTB 행동 간격: **2.50초** (SPD 120)
> SP 요구량: **4** (팀 공유)
> 액티브 쿨타임: **3턴**

**[전투 시작]** 「악마의 축복」 자동 발동
  → `스탯 변경` cri_ratio ↑15% (2턴) → 아군 전체 / `스탯 변경` atk ↑10%% (2턴) → 아군 전체

**[T1]** (행동시간 2.50s — **팀 내 가장 빠른 행동**) 「악마의 야망」 사용
  → 모든 아군보다 먼저 행동. 기본 공격으로 전장 정리 시작.

**[T2]** 「발푸르기스의 밤」 발동 (핵심 버프/디버프)
  → 팀 전체에 버프 부여 또는 적에게 디버프. 이후 아군 딜러의 공격이 강화된 상태로 타격.

**[T3]** 「악마의 야망」 반복
  → 빠른 SPD로 다시 행동. 적 위협 제거 또는 추가 지원.

**[T4]** (SP 4 충족 시) 「드림 오브 레기온」 발동 → **엑스트라 턴**
  → 팀 전체 강화 또는 적 전체 약화. 엑스트라 턴에서 액티브 재사용으로 이중 버프.

**[T5]** 버프 갱신 사이클
  → 2턴짜리 버프가 만료되는 시점. 액티브 재발동으로 버프 유지.

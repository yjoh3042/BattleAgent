# 그릴라 — 스킬 컨셉 상세 기획서

## 캐릭터 아이덴티티
| 항목 | 값 |
|---|---|
| 속성 | Forest |
| 직업 | Supporter |
| 성급 | 3★ |

## 스킬 상세

### 노멀
- 설명: 적 1명 대미지
- 강화+1: 대미지 +10%
- 강화+2: 적중 시 아군 1명 방어력 소폭 증가

### 액티브
- 설명: 목속성 아군 흡혈 버프 부여
- 강화+1: 흡혈 비율 +10%
- 강화+2: 목속성 아군 추가로 방어력 +10%

### 얼티밋
- 설명: 적 전체 공격력 감소 및 아군 전체 방어력 증가
- 강화+1: 대미지 +20%
- 강화+2: 목속성 아군 3턴간 방어력 +20% 추가 버프

### 패시브
- 설명: 아군 사망시 SP+1
- 강화+1: SP 획득 확률 5% → 10%
- 강화+2: 목속성 아군 디버프 지속 -1턴 감소

## 속성 상성
- 유리: 수속성
- 불리: 화염속성

## 핵심 포인트
- 강점: 팀 버프·시너지 — 방어력, 공격력
- 약점: 낮은 단독 전투력 — 아군 없이 무력

## 스킬 상세

### 노멀
- 설명: 적 1명 대미지
- 강화+1: 대미지 +10%
- 강화+2: 30% 확률로 화상 부여

### 액티브
- 설명: 적 1명 대미지 + 화상 2회 부여
- 강화+1: 대미지 +15%
- 강화+2: 화상 중인 적에게 화상 스택 +1 추가

### 얼티밋
- 설명: 적 전체 대미지 + 화상 2회 부여
- 강화+1: 대미지 +20%, 화상 스택 +1 추가
- 강화+2: 화상 중인 적 처치 시 SP +1

### 패시브
- 설명: 화상 중인 적 사망시 자신 얼티밋 쿨타임 초기화
- 강화+1: 쿨타임 감소량 +1
- 강화+2: 화상 중인 적 처치 시 아군 전체 공격력 +10% 2턴

## 속성 상성
- 유리: 목속성
- 불리: 수속성

## 핵심 포인트
- 강점: 고배율 공격 — 화상, 전체 대미지
- 약점: 낮은 방어력 — 집중 공격에 취약

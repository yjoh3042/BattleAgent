| 항목 | 값 |
|---|---|
| 속성 | Dark |
| 직업 | Healer |
| 성급 | 3★ |

## 스킬 상세

### 노멀
- 설명: 적 1명 대미지
- 강화+1: 대미지 +10%
- 강화+2: 적중 시 아군 최저HP 소량 회복

### 액티브
- 설명: 체력이 낮은 아군 1명 회복 + 출혈 중인 적 1명 대미지
- 강화+1: 회복량 +15%
- 강화+2: 회복 대상에게 치명타 확률 +10% 2턴 부여

### 얼티밋
- 설명: 아군 전체 회복 + 적 전체에 출혈 부여
- 강화+1: 대미지 +20%, 출혈 지속 +1턴
- 강화+2: 출혈 중인 적이 있으면 회복량 +30% 추가

### 패시브
- 설명: 출혈 중인 적이 있을 때 회복량 증가
- 강화+1: 회복량 +15%
- 강화+2: 출혈 적 수만큼 회복량 +5% 증가

## 속성 상성
- 유리: 광속성
- 불리: 없음속성

## 핵심 포인트
- 강점: 팀 회복·정화 — 출혈
- 약점: 낮은 공격력 — 전투 기여는 회복에 의존

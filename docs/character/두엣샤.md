# 두엣샤 — 스킬 컨셉 상세 기획서

## 캐릭터 아이덴티티
| 항목 | 값 |
|---|---|
| 속성 | Dark |
| 직업 | Magician |
| 성급 | 2★ |

## 스킬 상세

### 노멀
- 설명: 적 1명 대미지
- 강화+1: 대미지 +10%
- 강화+2: 치명타 시 추가 대미지 +15%

### 액티브
- 설명: 자신 치명타 확률 증가
- 강화+1: 버프 수치 +10% 증가
- 강화+2: 치명타 확률 중첩 최대치 +1 증가

### 얼티밋
- 설명: 적 전체 대미지 + 출혈
- 강화+1: 대미지 +20%, 출혈 지속 +1턴
- 강화+2: 침묵 또는 출혈 중인 적에게 추가 대미지 +30%

### 패시브
- 설명: 아군이 적의 버프 개수를 계산할 때 디버프도 버프로 취급 (최대 2개)
- 강화+1: 디버프 버프 취급 최대 2개 → 3개
- 강화+2: 취급된 디버프 1개당 자신 치명타 대미지 +5%

## 속성 상성
- 유리: 광속성
- 불리: 없음속성

## 핵심 포인트
- 강점: 광역 CC/디버프 — 출혈, 치명타, 전체 대미지
- 약점: 낮은 HP/DEF — 선공을 잡히면 위험

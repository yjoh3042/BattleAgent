# 살마키스 — 스킬 컨셉 상세 기획서

## 캐릭터 아이덴티티
| 항목 | 값 |
|---|---|
| 속성 | Fire |
| 직업 | Supporter |
| 성급 | 3★ |

## 스킬 상세

### 노멀
- 설명: 적 1명 대미지
- 강화+1: 대미지 +10%
- 강화+2: 적중 시 아군 1명 공격력 소폭 증가

### 액티브
- 설명: 적 1명 대미지 + 공격형 아군 공격력 증가 버프
- 강화+1: 대미지 +15%
- 강화+2: 화속성 아군 추가로 공격력 +10%

### 얼티밋
- 설명: 적 전체 대미지 + 화상 중인 적 수 비례 아군 전체 공격력 증가
- 강화+1: 대미지 +20%, 화상 스택 +1 추가
- 강화+2: 화속성 아군 3턴간 공격력 +20% 추가 버프

### 패시브
- 설명: 아군이 화상을 부여할 때마다 해당 아군 공격력 소폭 증가
- 강화+1: 추가 대미지 수치 +15%
- 강화+2: 화속성 아군 버프 지속 +1턴 증가

## 속성 상성
- 유리: 목속성
- 불리: 수속성

## 핵심 포인트
- 강점: 팀 버프·시너지 — 화상, 공격력, 전체 대미지
- 약점: 낮은 단독 전투력 — 아군 없이 무력

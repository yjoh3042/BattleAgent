# 비루파 — 스킬 컨셉 상세 기획서

## 캐릭터 아이덴티티
| 항목 | 값 |
|---|---|
| 속성 | Water |
| 직업 | Attacker |
| 성급 | 2★ |

## 스킬 상세

### 노멀
- 설명: 적 1명 대미지
- 강화+1: 대미지 +10%
- 강화+2: 30% 확률로 속도 감소

### 액티브
- 설명: 적 1명 3타 대미지 + 속도 감소
- 강화+1: 타격 횟수 +1
- 강화+2: 빙결 중인 적 대상 대미지 2배

### 얼티밋
- 설명: 적 1명 5타 대미지
- 강화+1: 타격 횟수 +2
- 강화+2: 빙결 중인 적 대상 방어력 무시 대미지

### 패시브
- 설명: 자신 HP 50% 이상일 때 타수마다 속도 증가, 50% 이하일 때 타수마다 흡혈
- 강화+1: 흡혈 비율 +5%
- 강화+2: HP 30% 이하 시 타수마다 치명타 확률도 증가

## 속성 상성
- 유리: 화염속성
- 불리: 목속성

## 핵심 포인트
- 강점: 고배율 공격 — 속도
- 약점: 낮은 방어력 — 집중 공격에 취약

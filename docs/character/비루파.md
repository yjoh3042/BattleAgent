## 3. 설계 의도

**기본 역할 설계**: 높은 ATK와 공격 배율로 적을 빠르게 처치하는 메인 딜러. 팀의 화력 핵심이며, 서포터/힐러의 보호 아래 최대 딜을 뽑아내는 것이 목표.

**고유 메커니즘**:

- **집중 타격**: 같은 대상을 반복 공격할수록 피해가 증가. 보스전이나 핵심 위협 제거에 특화된 싱글 타겟 킬러.

**트리거 설계 의도**:

- **ON_KILL**: 적 처치 시 패시브 발동 (1회 한정 보험). 마무리를 넣을수록 추가 행동/피해가 발생하여 스노우볼을 굴리는 설계.

### 밸런스 변경 이력

- **v8**: DAMAGE→DAMAGE_REPEAT_TARGET

## 4. 이상적 전투 흐름

> CTB 행동 간격: **3.75초** (SPD 80)
> SP 요구량: **6** (팀 공유)
> 액티브 쿨타임: **3턴**

**[T1]** (행동시간 3.75s) 「퇴마의 일격」 사용
  → 가장 가까운 적에게 기본 딜. 서포터/구속형이 먼저 움직여 디버프를 걸어둔 상태가 이상적.

**[T2]** (행동시간 7.50s) 「퇴마의 일격」 사용
  → 노말 반복으로 딜 축적. 액티브 쿨타임 대기.

**[T3]** (행동시간 11.25s) 「집중 강화」 사용 (액티브 쿨 해제)
  → 핵심 액티브 스킬 발동. 추가 효과(화상/방깎/CC 등)로 적 제압력 강화.

**[T4~5]** (SP 6 충족 시) 「파마의 일격」 발동 → **엑스트라 턴**
  → 얼티밋으로 최대 딜. 엑스트라 턴에서 노말/액티브 추가 사용.
  → 처치 성공 시 「집중의 일섬」 연쇄 발동 — **킬 체인 시작**

**[킬 체인]** 적 처치 → 패시브 추가 딜 → 다음 적 위협 → 연쇄 처치 가능
  → 이 캐릭터의 최대 포텐셜은 '첫 킬'에서 시작됨. 팀이 적 HP를 깎아놓으면 연쇄 처치로 전투를 결정.

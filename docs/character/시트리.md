# 시트리 — 스킬 컨셉 상세 기획서

## 캐릭터 아이덴티티
| 항목 | 값 |
|---|---|
| 속성 | Light |
| 직업 | Supporter |
| 성급 | 3★ |

## 스킬 상세

### 노멀
- 설명: 적 1명 대미지
- 강화+1: 대미지 +10%
- 강화+2: 적중 시 아군 1명 소량 보호막

### 액티브
- 설명: 체력이 낮은 아군 1명 보호막 부여
- 강화+1: 보호막 수치 +15%
- 강화+2: 보호막 중인 아군 추가로 공격력 +10%

### 얼티밋
- 설명: 모든 아군의 강화효과를 1턴씩 증가 시킴
- 강화+1: 효과 수치 +20%
- 강화+2: 버프 중인 아군 수만큼 전체 버프 지속 +1턴

### 패시브
- 설명: 턴 시작 시 자신에게 버프가 2개 이상이면 속도 +15
- 강화+1: 필요 버프 2개 → 1개
- 강화+2: 속도 증가 시 가장 가까운 아군에게도 속도 +5

## 속성 상성
- 유리: 암속성
- 불리: 없음속성

## 핵심 포인트
- 강점: 팀 버프·시너지 — 속도, 보호막
- 약점: 낮은 단독 전투력 — 아군 없이 무력

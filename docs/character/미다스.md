# 미다스 — 스킬 컨셉 상세 기획서

## 캐릭터 아이덴티티
| 항목 | 값 |
|---|---|
| 속성 | Forest |
| 직업 | Attacker |
| 성급 | 2★ |

## 스킬 상세

### 노멀
- 설명: 적 1명 대미지
- 강화+1: 대미지 +10%
- 강화+2: 30% 확률로 대상 방어력 감소

### 액티브
- 설명: 적 1명 대미지 + 대상 방어력 감소
- 강화+1: 대미지 +15%
- 강화+2: 방어력 감소 중인 적 대상 추가 대미지 +20%

### 얼티밋
- 설명: 적 전열 대미지 + 자신 방어력 증가
- 강화+1: 대미지 +20%
- 강화+2: 수면 또는 방어력 감소 중인 적에게 추가 대미지 +30%

### 패시브
- 설명: 방어력이 감소된 적 공격 시 추가 대미지
- 강화+1: 추가 대미지 수치 +15%
- 강화+2: 방어력 감소 적 처치 시 자신 방어력 +10% 2턴

## 속성 상성
- 유리: 수속성
- 불리: 화염속성

## 핵심 포인트
- 강점: 고배율 공격 — 방어력
- 약점: 낮은 방어력 — 집중 공격에 취약

# 야나 — 스킬 컨셉 상세 기획서

## 캐릭터 아이덴티티
| 항목 | 값 |
|---|---|
| 속성 | Light |
| 직업 | Attacker |
| 성급 | 1★ |

## 스킬 상세

### 노멀
- 설명: 적 1명 대미지
- 강화+1: 대미지 +10%
- 강화+2: 30% 확률로 대상 버프 1개 제거

### 액티브
- 설명: 적 1명 대미지 + 보호막 무시 대미지
- 강화+1: 보호막 수치 +15%
- 강화+2: 보호막이 있는 적 대상 보호막 무시 대미지

### 얼티밋
- 설명: 적 전열 대미지
- 강화+1: 대미지 +20%
- 강화+2: 버프가 없는 적에게 추가 대미지 +30%

### 패시브
- 설명: 자신 HP 80% 이상일 때 버프 있는 적 공격 시 대미지 증가
- 강화+1: HP 조건 80% → 70%
- 강화+2: HP 100%일 때 추가로 버프 1개 제거

## 속성 상성
- 유리: 암속성
- 불리: 없음속성

## 핵심 포인트
- 강점: 고배율 공격 — 보호막
- 약점: 낮은 방어력 — 집중 공격에 취약

# 맘몬 — 스킬 컨셉 상세 기획서

## 캐릭터 아이덴티티
| 항목 | 값 |
|---|---|
| 속성 | Forest |
| 직업 | Defender |
| 성급 | 2★ |

## 스킬 상세

### 노멀
- 설명: 적 1명 대미지
- 강화+1: 피격 시 자힐 5%
- 강화+2: 적중 시 자신 방어력 소폭 증가

### 액티브
- 설명: 적 1명에게 방어력 비례 대미지 + 넉백
- 강화+1: 대미지 +15%
- 강화+2: 사용 후 2턴간 받는 대미지 -15%

### 얼티밋
- 설명: 적 전열에 방어력 비례 대미지를 주고 도발
- 강화+1: 도발 지속 +1턴
- 강화+2: 사용 후 3턴간 아군 전체 방어력 +15%

### 패시브
- 설명: 턴 시작 시 목속성 아군 수에 비례해서 방어력 증가
- 강화+1: 아군 1명당 방어력 증가 수치 +3%
- 강화+2: 목속성 아군 3명 이상이면 추가로 반격 확률 +20%

## 속성 상성
- 유리: 수속성
- 불리: 화염속성

## 핵심 포인트
- 강점: 도발·반격 생존 — 방어력, 도발
- 약점: 낮은 화력 — 딜 기여도 낮음

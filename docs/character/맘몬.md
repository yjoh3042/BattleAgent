## 4. 이상적 전투 흐름

> CTB 행동 간격: **2.73초** (SPD 110)
> SP 요구량: **3** (팀 공유)
> 액티브 쿨타임: **3턴**

**[T1]** (행동시간 2.73s — SPD 110으로 딜러보다 먼저 행동) 「대지의 주먹」 사용
  → 기본 공격으로 어그로 생성. 전열 배치로 적 공격을 자연스럽게 유도.

**[T2]** 「지진 강타」 발동 (핵심)
  → 도발/보호막/반격 준비 등 방어 체계 구축. 이 시점부터 팀 방어막 역할 시작.

**[T3~4]** 「대지의 주먹」 반복 + 피격 유도
  → 도발 상태에서 적 공격을 받으며 반격/패시브 발동으로 딜 기여.
  → 피격 시 「대지의 반격」 발동 — 반격 + 추가 효과

**[T4~5]** (SP 3 충족 시) 「대지 분쇄」 발동 → **엑스트라 턴**
  → 팀 보호 얼티밋으로 결정적 순간에 팀을 지킴.

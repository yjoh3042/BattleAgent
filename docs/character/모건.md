# 모건 — 스킬 컨셉 상세 기획서

## 캐릭터 아이덴티티
| 항목 | 값 |
|---|---|
| 속성 | Fire |
| 직업 | Magician |
| 성급 | 3★ |

## 스킬 상세

### 노멀
- 설명: 적 1명 대미지
- 강화+1: 대미지 +10%
- 강화+2: 30% 확률로 화상 부여

### 액티브
- 설명: 자신에게 공격력 증가, 명중률 증가 버프
- 강화+1: 버프 수치 +10% 증가
- 강화+2: 화상 중인 적에게 치명타 확률 +30%

### 얼티밋
- 설명: 무작위 적 3명 대미지 + 대상에게 걸린 버프 2개씩 제거(최근에 걸린 순)
- 강화+1: 제거 개수 +1 추가
- 강화+2: 화상 중인 적 수만큼 전체 대미지 +10% 증가

### 패시브
- 설명: 버프 중인 적 공격시 추가대미지
- 강화+1: 추가 대미지 수치 +15%
- 강화+2: 화상 부여 시 30% 확률로 화상 스택 +1 추가

## 속성 상성
- 유리: 목속성
- 불리: 수속성

## 핵심 포인트
- 강점: 광역 CC/디버프 — 공격력
- 약점: 낮은 HP/DEF — 선공을 잡히면 위험

# 유나 — 스킬 컨셉 상세 기획서

## 캐릭터 아이덴티티
| 항목 | 값 |
|---|---|
| 속성 | Dark |
| 직업 | Supporter |
| 성급 | 3★ |

## 스킬 상세

### 노멀
- 설명: 적 1명 대미지
- 강화+1: 대미지 +10%
- 강화+2: 적중 시 아군 1명 치명타 확률 소폭 증가

### 액티브
- 설명: 적 1명 대미지 + 아군 전체 치명타 확률 증가
- 강화+1: 대미지 +15%
- 강화+2: 암속성 아군 추가로 치명타 대미지 +15%

### 얼티밋
- 설명: 적 전체 대미지 + 아군 전체 치명타 대미지 증가
- 강화+1: 타격 횟수 +2
- 강화+2: 암속성 아군 3턴간 치명타 확률 +15% 추가 버프

### 패시브
- 설명: 아군 치명타 발생 시 해당 아군의 다음 공격 대미지 증가
- 강화+1: 추가 대미지 수치 +15%
- 강화+2: 치명타 발생 시 해당 아군 디버프 1개 제거

## 속성 상성
- 유리: 광속성
- 불리: 없음속성

## 핵심 포인트
- 강점: 팀 버프·시너지 — 치명타, 전체 대미지
- 약점: 낮은 단독 전투력 — 아군 없이 무력

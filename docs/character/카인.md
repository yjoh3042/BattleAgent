# 카인 — 스킬 컨셉 상세 기획서

| 항목 | 값 |
|---|---|

## 스킬 상세

### 노멀
- 설명: 적 1명 대미지
- 강화+1: 대미지 +10%
- 강화+2: 30% 확률로 화상 부여

### 액티브
- 설명: 자신의 체력 감소 + 공격력 증가 버프(중첩 가능)
- 강화+1: 버프 수치 +10% 증가
- 강화+2: 적중 시 화상 중인 대상 추가 대미지 +20%

### 얼티밋
- 설명: 적 전체 대미지
- 강화+1: 대미지 +20%
- 강화+2: 화상 중인 적 처치 시 SP +1

### 패시브
- 설명: 아군 사망 시 자신 공격력 대폭 증가 (영구)
- 강화+1: 공격력 증가 수치 +10%
- 강화+2: 사망 아군이 2명 이상이면 추가로 치명타 확률 +30%

## 속성 상성
- 유리: 목속성
- 불리: 수속성

## 핵심 포인트
- 강점: 고배율 공격 — 공격력, 전체 대미지
- 약점: 낮은 방어력 — 집중 공격에 취약

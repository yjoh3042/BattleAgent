# 드미테르 — 스킬 컨셉 상세 기획서

## 캐릭터 아이덴티티
| 항목 | 값 |
|---|---|
| 속성 | Fire |
| 직업 | Supporter |
| 성급 | 2★ |

## 스킬 상세

### 노멀
- 설명: 적 1명 대미지
- 강화+1: 대미지 +10%
- 강화+2: 적중 시 아군 1명 공격력 소폭 증가

### 액티브
- 설명: 전열의 적에게 대미지 + 방어력 감소 부여
- 강화+1: 대미지 +15%
- 강화+2: 화속성 아군 추가로 공격력 +10%

### 얼티밋
- 설명: 적 1명에게 대미지 + 디버프가 가장 많은 아군 1명의 디버프 모두 제거
- 강화+1: 제거 개수 +1 추가
- 강화+2: 화속성 아군 3턴간 공격력 +20% 추가 버프

### 패시브
- 설명: 아군의 디버프가 해제 될때 마다 회복
- 강화+1: 회복량 +15%
- 강화+2: 화속성 아군 버프 지속 +1턴 증가

## 속성 상성
- 유리: 목속성
- 불리: 수속성

## 핵심 포인트
- 강점: 팀 버프·시너지 — 방어력
- 약점: 낮은 단독 전투력 — 아군 없이 무력

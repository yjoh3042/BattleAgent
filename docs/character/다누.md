# 다누 — 스킬 컨셉 상세 기획서

## 캐릭터 아이덴티티
| 항목 | 값 |
|---|---|
| 속성 | Forest |
| 직업 | Attacker |
| 성급 | 3★ |

## 스킬 상세

### 노멀
- 설명: 적 1명 대미지
- 강화+1: 대미지 +10%
- 강화+2: 30% 확률로 대상 방어력 감소

### 액티브
- 설명: 적 1명 대미지 + 자신 방어력 비례 추가 대미지
- 강화+1: 대미지 +15%
- 강화+2: 방어력 감소 중인 적 대상 추가 대미지 +20%

### 얼티밋
- 설명: 적 전열 대미지 + 자신 방어력 증가 버프
- 강화+1: 대미지 +20%
- 강화+2: 수면 또는 방어력 감소 중인 적에게 추가 대미지 +30%

### 패시브
- 설명: 피격 시 방어력 버프 자동 부여 (3턴 쿨타임)
- 강화+1: 쿨타임 3턴 → 2턴
- 강화+2: 방어력 버프 중 공격 시 추가 대미지 +20%

## 속성 상성
- 유리: 수속성
- 불리: 화염속성

## 핵심 포인트
- 강점: 고배율 공격 — 방어력
- 약점: 낮은 방어력 — 집중 공격에 취약

# 다이아나 — 스킬 컨셉 상세 기획서

## 캐릭터 아이덴티티
| 항목 | 값 |
|---|---|
| 속성 | Forest |
| 직업 | Magician |
| 성급 | 1★ |

## 스킬 상세

### 노멀
- 설명: 적 1명 대미지
- 강화+1: 대미지 +10%
- 강화+2: 30% 확률로 수면 부여

### 액티브
- 설명: 적 전체에 중독 부여
- 강화+1: 중독 스택 +1 추가
- 강화+2: 중독 2스택 이상인 적 속도 -10

### 얼티밋
- 설명: 적 전체 대미지 + 중독 중인 적 50% 확률로 수면
- 강화+1: 대미지 +20%, 수면 확률 +15%
- 강화+2: 중독+수면 동시 적에게 3배 대미지

### 패시브
- 설명: 중독 + 수면이 동시에 걸린 적에게 대미지 증가
- 강화+1: 추가 대미지 수치 +15%
- 강화+2: 중독+수면 적 처치 시 주변 적에게 중독 전이

## 속성 상성
- 유리: 수속성
- 불리: 화염속성

## 핵심 포인트
- 강점: 광역 CC/디버프 — 수면, 중독, 전체 대미지
- 약점: 낮은 HP/DEF — 선공을 잡히면 위험

## 3. 설계 의도

**기본 역할 설계**: 빠른 SPD(100)로 선제 행동하여 CC와 디버프로 적을 제압. 딜보다는 적 행동 방해와 전장 통제에 특화된 컨트롤러.

**고유 메커니즘**:

- **디버프 전이**: 한 적에게 걸린 디버프를 주변 적에게 확산. 소수 타겟 디버프를 전체 디버프로 전환하는 효율적 CC 확장.

**트리거 설계 의도**:

- **ON_KILL**: 적 처치 시 패시브 발동 (1회 한정 보험). 마무리를 넣을수록 추가 행동/피해가 발생하여 스노우볼을 굴리는 설계.

## 4. 이상적 전투 흐름

> CTB 행동 간격: **3.00초** (SPD 100)
> SP 요구량: **4** (팀 공유)
> 액티브 쿨타임: **3턴**

**[T1]** (행동시간 3.00s — 서포터 다음으로 빠름) 「목동의 휘파람」 사용
  → 선제 공격 + CC/디버프 부여. 적 딜러보다 먼저 행동하여 위협 제거.

**[T2]** 「목동의 휘파람」 반복
  → CC 재적용 또는 디버프 유지. 적의 행동 기회를 계속 줄임.

**[T3]** 「작은 양을 위한 굴레」 발동 (쿨타임 해제)
  → 핵심 CC/디버프 콤보. 적 핵심 유닛을 무력화.

**[T4]** (SP 4 충족 시) 「고요한 봄의 들판」 발동 → **엑스트라 턴**
  → AoE CC/디버프로 적 전체 제압. 엑스트라 턴에서 노말로 추가 CC.
  → 처치 시 「독의 추격」 연쇄 발동

# 판 — 스킬 컨셉 상세 기획서

## 캐릭터 아이덴티티
| 항목 | 값 |
|---|---|
| 속성 | Forest |
| 직업 | Magician |
| 성급 | 3★ |

## 스킬 상세

### 노멀
- 설명: 적 1명 대미지
- 강화+1: 대미지 +10%
- 강화+2: 30% 확률로 수면 부여

### 액티브
- 설명: 적 1명 대미지 및 수면부여
- 강화+1: 대미지 +15%
- 강화+2: 수면 중인 적에게 방어력 무시 대미지

### 얼티밋
- 설명: 적 전체 대미지 및 후열의 적 수면 부여
- 강화+1: 대미지 +20%, 수면 지속 +1턴
- 강화+2: 수면 중인 적 수만큼 전체 대미지 +15% 증가

### 패시브
- 설명: 판의 공격은 수면 부여 후 2턴 이내 대상을 공격해도 깨우지 않음
- 강화+1: 수면 불해제 기간 2턴 → 3턴
- 강화+2: 수면 중인 적 공격 시 수면 지속 +1턴 (최대 3턴)

## 속성 상성
- 유리: 수속성
- 불리: 화염속성

## 핵심 포인트
- 강점: 광역 CC/디버프 — 수면, 전체 대미지
- 약점: 낮은 HP/DEF — 선공을 잡히면 위험

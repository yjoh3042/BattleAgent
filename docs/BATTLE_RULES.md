# BattleAgent 전투 룰 상세 명세서

> **v8.0** | CTB 턴제 5v5 자동 전투 시스템

---

## 1. 전투 구조

### 1.1 기본 설정

| 항목 | 값 | 설명 |
|------|---|------|
| 팀 구성 | 5 vs 5 | 아군/적군 각 5명 |
| 배치 | 3×3 그리드 | row 0=전열, 2=후열 / col 0=좌, 2=우 |
| 최대 턴 | 200턴 | 초과 시 타임오버 |
| 타임라인 한도 | 300.0 | 초과 시 타임오버 |
| 라운드 간격 | 10.0 (타임라인 단위) | 매 10.0마다 라운드 전환 |

### 1.2 승리 조건

| 조건 | 결과 |
|------|------|
| 적군 전멸 (HP ≤ 0) | 아군 승리 |
| 아군 전멸 (HP ≤ 0) | 적군 승리 |
| 타임오버 + 공격덱(OFFENSE) | 적군 승리 (전멸 실패) |
| 타임오버 + 방어덱(DEFENSE) | 아군 승리 (생존 성공) |

### 1.3 라운드 전환

타임라인이 10.0 단위를 넘을 때마다 라운드 전환 발생:

1. 양측 SP → 0으로 리셋
2. 전체 유닛 `used_ultimate_this_round` 플래그 리셋
3. 전체 유닛 행동 큐 재계산 (현재 SPD 기준)
4. `ON_ROUND_START` 트리거 평가

## 2. CTB 턴 시스템

### 2.1 행동 시간 계산

```
행동시간 = 현재시간 + (300 / SPD)
```

- 초기 행동시간: `300 / SPD` (SPD가 높을수록 먼저 행동)
- 예시: SPD 120 → 2.5초, SPD 80 → 3.75초

### 2.2 역할별 기본 SPD

| 역할 | SPD | 행동 주기 |
|------|-----|-----------|
| 서포터 | 120 | 2.50 (가장 빠름) |
| 탱커 | 110 | 2.73 |
| 마법사 | 100 | 3.00 |
| 힐러 | 90 | 3.33 |
| 딜러 | 80 | 3.75 (가장 느림) |

### 2.3 SPD 변동 시 재계산

버프/디버프로 SPD가 변할 때:
```
남은거리 = (기존_다음행동시간 - 현재시간) × 기존SPD
새_다음행동시간 = 현재시간 + 남은거리 / 새SPD
```
- 최소 SPD: 25.0 (이하로 내려가지 않음)

### 2.4 엑스트라 턴 (얼티밋)

- 삽입 시간: `현재시간 - ε × (엑스트라_횟수 + 1)` (ε = 0.000001)
- 즉시 행동 보장 (현재 턴 직후)
- 엑스트라 턴 최대: 100회 (무한루프 방지)
- 엑스트라 턴에서는 SP 충전, 쿨타임 감소, 턴 종료 처리 **없음**

## 3. 턴 진행 순서

```
1. CTB 큐에서 다음 유닛 Pop
2. 현재 시간 갱신
3. SP 충전 (+1 양측, 엑스트라 턴·SP잠금 제외)
4. ──── 턴 시작 페이즈 ────
   a. DoT 피해 (화상/중독/출혈)
   b. HoT 회복 (재생)
   c. CharacterTurnStart 버프 틱 (duration -1)
   d. 만료 버프 제거
5. ──── CC 판정 ────
   a. Hard CC (기절/빙결/수면/석화) → 행동 불가, 턴 종료로 건너뜀
   b. Soft CC (감전/공포) → 30% 확률로 행동 실패
6. ──── 얼티밋 우선 체크 ────
   a. SP 충분 + 쿨타임 0 + 라운드 미사용 → 엑스트라 턴 등록
7. ──── 스킬 결정 ────
   a. 엑스트라 턴 → 얼티밋 강제 실행
   b. 혼란/침묵 → 노말 스킬 강제
   c. 액티브 쿨타임 0 → 액티브 스킬
   d. 그 외 → 노말 스킬
8. ──── 스킬 실행 ────
   a. 각 SkillEffect에 대해 타겟 선택 → 효과 적용
   b. 처치 발생 시 ON_KILL 트리거
9. ──── 얼티밋 사용 트리거 ────
   a. 엑스트라 턴이었으면 ON_ULTIMATE_USED 평가
10. ──── 턴 종료 페이즈 ────
    a. CharacterTurnEnd 버프 틱 (duration -1)
    b. CC 틱 (duration -1)
    c. 도발 틱 (duration -1)
    d. 액티브 쿨타임 -1
    e. 얼티밋 쿨타임 -1
    f. ON_TURN_END 트리거
    g. 유닛 재스케줄 (다음 행동시간 등록)
11. ──── 승패 판정 ────
```

## 4. 데미지 공식

### 4.1 기본 데미지

```
유효방어력 = max(0, DEF - 관통)
기본피해 = ATK × (1 - 유효방어력 / (ATK + 유효방어력))
```

### 4.2 최종 피해 계산

```
최종피해 = floor(기본피해 × 스킬배율 × 속성보정 × 크리배율 × 화상보너스)
```

### 4.3 속성 상성 배율

```
화 → 목: 1.2×    목 → 수: 1.2×    수 → 화: 1.2×
광 → 암: 1.2×    암 → 광: 1.2×
역방향: 0.8×     동일/무관: 1.0×
```

### 4.4 크리티컬

```
유효크리율 = max(0, 크리율 - 크리저항)
크리 발동: random() < 유효크리율
크리 배율: 크리피해율 (기본 1.5 = 150%)
```

| 기본값 | 수치 |
|--------|------|
| 크리율 | 15% |
| 크리피해 | 150% |
| 크리저항 | 0% |

### 4.5 회피

```
유효회피율 = max(0, 회피 - (1 - 명중))
회피 발동: random() < 유효회피율
```
- 기본 회피: 0%, 기본 명중: 100%

### 4.6 데미지 변형

| 타입 | 공식 | 비고 |
|------|------|------|
| DAMAGE | 기본 공식 | 표준 |
| DAMAGE_PENETRATION | ATK × 배율 × 속성 × 크리 | DEF 완전 무시 |
| DAMAGE_HP_RATIO | max(1, target.maxHP × 비율) | DEF 무관 |
| DAMAGE_CRI | 기본 공식 + 강제 크리 | 항상 크리 적용 |
| DAMAGE_BUFF_SCALE | 기본 × (1 + 버프수 × 0.1) | 시전자/타겟 버프 수 |
| DAMAGE_DEBUFF_SCALE_TARGET | 기본 × (1 + 디버프수 × 0.1) | 타겟 디버프 수 |
| DAMAGE_BURN_BONUS | 화상 시 배율×2 + 강제크리 / 없으면 기본 | 화상 시너지 |
| DAMAGE_ESCALATE | 기본 × (1 + 0.2 × 사용횟수) | 전투 중 누적 |
| DAMAGE_REPEAT_TARGET | 기본 × (1 + 0.3 × 반복횟수) | 같은 적 집중 |
| DAMAGE_MISSING_HP_SCALE | 기본 × (1 + (1-HP비율) × 배율) | 잃은HP 비례 |

### 4.7 화상 보너스

```
화상보너스 = 1.0 + (0.5 × 화상스택)
```
- 1스택: 1.5×, 2스택: 2.0×, 3스택: 2.5×

## 5. 회복 공식

| 타입 | 공식 |
|------|------|
| HEAL | 시전자ATK × 스킬배율 |
| HEAL_HP_RATIO | 대상maxHP × HP비율 |
| HEAL_LOSS_SCALE | (대상maxHP - 현재HP) × 비율 |
| HEAL_PER_HIT | 적 피해 적중 시 아군 최저HP 대상maxHP × 비율 |
| HEAL_CURRENT_HP_SCALE | (시전자ATK × 배율) × 시전자HP비율 |
| DOT_HEAL_HP_RATIO (HoT) | 대상maxHP × 재생비율 × 스택 |

## 6. DoT (지속 피해) 시스템

### 6.1 DoT 피해 공식

```
방어감쇄 = DEF / (DEF + 300)
DoT피해 = floor(max(1, maxHP × DoT비율 × 스택수 × (1 - 방어감쇄)))
```

### 6.2 DoT 종류

| 타입 | 최대 스택 | 발동 시점 | 비고 |
|------|-----------|-----------|------|
| 화상(burn) | 5 | 턴 시작 | 화상보너스 시너지 |
| 중독(poison) | 3 | 턴 시작 | 순수 지속피해 |
| 출혈(bleed) | 3 | 턴 시작 | 순수 지속피해 |

### 6.3 DoT 스택 규칙

- 같은 source_skill_id → 스택 증가 (최대치까지), 지속턴 갱신
- 다른 source_skill_id → 별도 DoT로 중복 적용
- 스택 소멸 시 해당 수만큼 태그 제거

## 8. 얼티밋 시스템

### 8.1 발동 조건

모든 조건 충족 시 발동:
1. 얼티밋 쿨타임 ≤ 0
2. 이번 라운드 미사용
3. 팀 SP ≥ 소모량
4. 엑스트라 턴 아닐 때 (이미 얼티밋 중이면 불가)

### 8.2 발동 순서

```
조건 충족 → SP 소모 → 엑스트라 턴 등록 → (다음 큐에서) 얼티밋 강제 실행
→ 쿨타임 4턴 시작 → 라운드 사용 플래그 ON
```

### 8.3 얼티밋 쿨타임

| 항목 | 값 |
|------|---|
| 쿨타임 | 4턴 (전역할 동일) |
| 감소 시점 | 턴 종료 시 -1 |
| 쿨타임 변경 | ACTIVE_CD_CHANGE 효과로 가능 |

## 9. 버프/디버프 시스템

### 9.1 적용 규칙

1. **디버프면역 체크**: `is_debuff_immune = True` → 디버프 차단
2. **중복 판정**:
   - 같은 source_skill_id + buff_id → 지속턴 갱신 (DoT는 스택 증가)
   - 다른 source_skill_id → 별도 버프로 중복 적용

### 9.2 스탯 버프 계산

```
비율 버프: 변화량 = 기본스탯 × 버프값 × 스택수
고정 버프: 변화량 = 버프값 × 스택수
최종 스탯 = 기본스탯 + 모든_버프_합계
```

### 9.3 틱 타이밍

| 시점 | 처리 내용 |
|------|-----------|
| CharacterTurnStart | DoT/HoT 피해/회복 → duration -1 → 만료 제거 |
| CharacterTurnEnd | duration -1 → 만료 제거 → CC/도발/쿨 틱 |

### 9.4 특수 상태 버프

| 상태 | 효과 | 마커 태그 |
|------|------|-----------|
| 무적(Invincibility) | 모든 피해 0 | `invincibility` |
| 불사(Undying) | HP 1 이하 불가 | `undying` |
| 디버프면역 | 디버프 차단 | `debuff_immune` |
| SP잠금 | SP 충전 차단 | `sp_lock` |
| 크리불가 | 크리티컬 불가 | `cri_unavailable` |
| 반격불가 | 반격 차단 | `counter_unavailable` |
| 속성무시 | 상성 배율 1.0 | `ignore_element` |

## 10. CC (상태이상) 시스템

### 10.1 Hard CC (행동 불가)

| CC | 효과 |
|----|------|
| 기절(STUN) | 행동 불가 |
| 수면(SLEEP) | 행동 불가 |
| 빙결(FREEZE) | 행동 불가 |
| 석화(STONE) | 행동 불가 |
| 스킬이상(ABNORMAL_SKILL) | 행동 불가 |

- Hard CC 시 DoT는 정상 발동 (턴 시작 틱)
- 행동 불가 → 턴 종료 처리로 직행

### 10.2 Soft CC (확률적 행동 실패)

| CC | 효과 |
|----|------|
| 감전(ELECTRIC_SHOCK) | 30% 행동 실패 |
| 공포(PANIC) | 30% 행동 실패 |

### 10.3 디버프형 CC

| CC | 효과 |
|----|------|
| 중독(POISON) | DoT (턴당 피해) |
| 화상(BURN) | DoT + 화상보너스 시너지 |
| 실명(BLIND) | 명중률 감소 |
| 침묵(SILENCE) | 액티브 스킬 사용 불가 → 노말 강제 |
| 혼란(CONFUSED) | 타겟 랜덤 변경 + 액티브 사용 불가 |

| ENEMY_ELEMENT_WEAK | 상성 약점 적 우선 (없으면 NEAR) |

### 11.3 타일 거리 공식

```
거리 = 시전자.row + 대상.row + |시전자.col - 대상.col|
```
- 양 진영이 마주보는 구조 (row 0끼리 가장 가까움)
- 동거리 시 HP 낮은 쪽 우선

### 11.4 도발 처리

- 도발 상태 시:
  - 단일 타겟 스킬 → 도발자로 강제 변경
  - 다중 타겟 스킬 → 도발자 반드시 포함
  - 아군 타겟 스킬 → 도발 무시

## 12. 보호막/무적/불사

### 12.1 피해 흡수 우선순위

```
1. 무적(Invincible) → 피해 0
2. 보호막(Barrier) → min(보호막HP, 피해) 흡수
3. 불사(Undying) → HP 1 미만 방지
4. 일반 HP 감소
```

### 12.2 보호막

| 타입 | 공식 |
|------|------|
| BARRIER | 시전자ATK × 스킬배율 |
| BARRIER_RATIO | 대상maxHP × 비율 |

## 13. 트리거 시스템

### 13.1 트리거 이벤트

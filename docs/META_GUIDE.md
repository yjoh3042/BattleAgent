# BattleAgent 메타 가이드 v8.2

## 시스템 개요
- CTB(Conditional Turn-Based) 시스템: 300/SPD로 행동순서 결정
- 속성: Fire, Water, Forest, Light, Dark (화>목>수>화, 광<>암)
- 직업: Attacker, Magician, Defender, Healer, Supporter
- SP: 파티 공유, 얼티밋 사용 시 소모 + 엑스트라턴

## 10대 메타팀

### M01 화상연소 [Fire] — DoT 누적→폭딜
- 구성: 카라라트리(딜) / 다비2(마법) / 살마키스(서폿) / 지바(힐) / 데레사(탱)
- 승리조건: 화상 스택 극대화 → 카라라트리 패시브(화상수 비례딜) 폭발
- 전투 흐름:
  - T1(세팅): 살마키스 공격력↑ / 데레사 반격준비
  - T2(착화): 카라라트리 전열화상 / 다비2 화상+전이
  - T3(유지): 지바 디버프제거+힐 / 데레사 반격화상
  - T4(폭발): 카라라트리 얼티밋 전체딜+화상크리 ★핵심턴
  - T5(마무리): 살마키스 얼티밋 화상수비례 팀공격력↑

### M02 빙결제어 [Water] — 턴 봉쇄→일방 학살
- 구성: 바리(마법) / 니르티(딜) / 도계화(마법) / 티스베(힐) / 상아(서폿)
- 승리조건: 속도감소→빙결 연쇄→적 행동불가 일방딜
- 전투 흐름:
  - T1: 상아 속도감소+빙결 / 바리 4타+속도감소
  - T2: 도계화 추가감속 / 바리 패시브 빙결발동
  - T3: 니르티 전열딜+빙결 / 티스베 회피율↑
  - T4: 바리 얼티밋 전체속도감소→빙결확산 ★
  - T5: 니르티 빙결대상 일방학살

### M03 수면폭발 [Forest] — CC중첩→폭딜
- 구성: 에레보스(마법) / 판(마법) / 다이아나(마법) / 아우로라(힐) / 그릴라(서폿)
- 승리조건: 중독+수면 축적→에레보스 수면2배딜
- T1: 다이아나 전체중독 / 그릴라 흡혈버프
- T2: 판 수면 / 다이아나 중독적 50% 수면전환
- T3: 에레보스 후열수면 / 판 2턴 수면불해제
- T4: 에레보스 얼티밋 수면2배딜 ★
- T5: 아우로라 전체힐+면역 지속력확보

### M04 치명타학살 [Dark] — 즉사+연쇄처형
- 구성: 아누비스(딜) / 바토리(서폿) / 유나(서폿) / 아르테미스(마법) / 페르세포네(서폿)
- 승리조건: 크리풀스택→아누비스 즉사+처치 공격력중첩
- T1: 유나 전체크리↑ / 바토리 직선딜+크리↑
- T2: 아르테미스 크리중첩 / 페르세포네 크리↑
- T3: 아누비스 HP25%↓ 즉사 / 바토리 SP+1 ★
- T4: 아누비스 얼티밋 전체+최저HP2배
- T5: 아르테미스 얼티밋 전체+침묵 봉쇄

### M05 속도압도 [Water] — 선제턴 독점→압살
- 구성: 아라한(서폿) / 이브(딜) / 리자(딜) / 에우로스(힐) / 레오(서폿)
- 승리조건: 속도극대화→선공→멀티히트 압도
- T1: 레오 회피율↑ / 에우로스 힐+속도↑
- T2: 아라한 딜+SP+1 / 이브 기절
- T3: 리자 다타격 공격력↑누적 / 아라한 속도비례딜
- T4: 이브 7타+출혈 / 리자 7타 ★
- T5: 아라한 얼티밋 전체속도↑ 선제독점

### M06 철벽수호 [Mixed] — 장기전→소모전 승리
- 구성: 맘몬(탱) / 메티스(탱) / 모나2(탱) / 브라우니(서폿) / 자청비(서폿)
- 승리조건: 방어력3중스택+보호막+디버프제거 난공불락
- T1: 메티스 방어력↑+보호막 / 자청비 방어력↑+지속힐
- T2: 모나2 보호막+도발 / 맘몬 목속성비례 방어력↑
- T3: 브라우니 매턴디버프제거 / 메티스 버프2↑ 피해감소
- T4: 맘몬 방어비례딜+도발 / 모나2 전체보호막 ★
- T5: 메티스 얼티밋 전체방어력↑ 3중스택완성

### M07 출혈암살 [Dark] — DoT+흡혈 자급자족
- 구성: 바토리(서폿) / 미르칼라(힐) / 두엣샤(마법) / 프레이(탱) / 쿠바바(딜)
- 승리조건: 출혈축적→미르칼라 출혈힐+쿠바바 쿨감 순환
- T1: 두엣샤 크리↑ / 바토리 직선+크리↑
- T2: 프레이 반격↑ / 쿠바바 크리시쿨감
- T3: 바토리+두엣샤 얼티밋 출혈부여 ★
- T4: 미르칼라 힐+출혈적딜 / 출혈회복↑
- T5: 쿨감으로 스킬재사용 무한루프

### M08 보호막연합 [Light] — 불멸 지구전
- 구성: 루미나(마법) / 티타니아(탱) / 오네이로이(서폿) / 다나(힐) / 시트리(서폿)
- 승리조건: 보호막다중부여+디버프면역/제거 무적유지
- T1: 시트리 보호막 / 오네이로이 보호막+버프연장
- T2: 티타니아 방어력↑+디버프면역 / 다나 회복
- T3: 루미나 딜+디버프제거 / 오네이로이 디버프↓
- T4: 루미나+티타니아 얼티밋 이중보호막 ★
- T5: 시트리 버프지속+1턴 풀스택영구유지

### M09 디버프착취 [Mixed] — 축적→일괄폭발
- 구성: 아슈토레스(딜) / 드미테르(서폿) / 루미나(마법) / 카인(딜) / 모건(마법)
- 승리조건: 디버프 다량축적→아슈토레스 디버프폭발 섬멸
- T1: 모건 공격력↑ / 카인 HP소모+공격력↑중첩
- T2: 드미테르 전열+방감 / 아슈토레스 전열딜
- T3: 루미나 딜+아군디버프제거 / 드미테르 회복
- T4: 아슈토레스 얼티밋 디버프폭발 ★핵심턴
- T5: 모건 얼티밋 버프제거 적 체계붕괴

### M10 혼성엘리트 [Mixed] — 올라운드 만능
- 구성: 아누비스(딜) / 에레보스(마법) / 카라라트리(딜) / 루미나(마법) / 아우로라(힐)
- 승리조건: 3.5★ 3명 개별파워+속성분산 만능대응
- T1: 아우로라 계약힐 / 루미나 디버프제거
- T2: 카라라트리 화상 / 에레보스 수면
- T3: 아누비스 즉사 / 루미나 공격력↑
- T4: 에레보스+카라라트리 얼티밋 러시 ★
- T5: 아누비스 얼티밋 최저HP2배 마무리

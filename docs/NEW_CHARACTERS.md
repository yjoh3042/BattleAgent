## NC01: 아테나 (Athena)

### 기본 정보

| 항목   | 값                    |
|--------|-----------------------|
| 속성   | Light (광)            |
| 직업   | Magician              |
| 성급   | 4★                   |
| ATK    | 950                   |
| DEF    | 400                   |
| HP     | 4000                  |
| SPD    | 92                    |
| 행동간격 | 3.26 (300/92)       |

> 4★ 스탯은 3.5★ 기준 +약 12% 보정 적용.

### 스킬 상세

**[노멀] 심판의 빛**
- 효과: 적 1명에게 ATK × 1.0 광속성 대미지

**[액티브] 정화의 심판 (강화 전)**
- 효과: 적 전열 전체에게 ATK × 2.2 광속성 대미지 + 40% 확률로 기절 (1턴) + 아군 디버프 1개 제거

**[액티브] 정화의 심판 +1**
- 효과: 적 전열 전체에게 ATK × 2.2 광속성 대미지 + 50% 확률로 기절 (1턴) + 아군 디버프 2개 제거
- 강화 추가: 기절 확률 +10%, 정화 대상 +1

**[액티브] 정화의 심판 +2**
- 효과: 적 전열 전체에게 ATK × 2.4 광속성 대미지 + 50% 확률로 기절 (2턴) + 아군 전체 디버프 1개 제거
- 강화 추가: 기절 지속 1턴 증가, 정화 범위 아군 전체로 확장

**[얼티밋] 최후의 심판 (강화 전)**
- 효과: 적 전체에게 ATK × 4.2 광속성 대미지 + 디버프 보유 적에게 ATK × 1.5 추가 대미지 + 아군 전체 디버프 전부 제거

**[얼티밋] 최후의 심판 +1**
- 효과: 적 전체에게 ATK × 4.2 광속성 대미지 + 디버프 보유 적에게 ATK × 2.0 추가 대미지 + 아군 전체 디버프 전부 제거
- 강화 추가: 추가 대미지 배율 +0.5

**[얼티밋] 최후의 심판 +2**
- 효과: 적 전체에게 ATK × 4.2 광속성 대미지 + 디버프 보유 적에게 ATK × 2.0 추가 대미지 + 아군 전체 디버프 전부 제거 + 이번 턴 적에게 적용된 디버프 수 × 5% 추가 대미지
- 강화 추가: 디버프 스택 수 비례 폭딜 기믹 추가

**[패시브] 심판의 낙인**
- 효과: 아테나가 적에게 기절을 부여할 때, 해당 적의 버프를 전부 제거

### 설계 의도

광속성 캐릭터는 현재 고대미지 또는 힐/보호 계열로만 분포. 적을 제어하는 수단이 전혀 없어 Light 파티는 선공 고화력 이외 전략이 부재했다. 아테나는 광속성 최초 CC 보유자로서 Light 파티에 제어-정화 전략 축을 추가한다.

### 메타 영향 분석

- **긍정적**: Light 파티의 전략 다양성 확장. 기존 Dark 파티(버프 활용)에 강력한 카운터 등장.
- **주의점**: 전열 기절 50% + 버프 제거 패시브 조합이 PvP에서 과도할 수 있음. 기절 확률 모니터링 필요.
- **DPS 확인**: ATK 950 × 4.2 = 3990 + 추가 최대 1900 = 5890 (얼티밋 1회). 단발 최대치로 정상 범위.

### 시너지 파트너

| 캐릭터 유형 | 시너지 이유 |
|-------------|-------------|
| Dark 디버퍼 | 적에게 디버프 누적 → 아테나 얼티밋 추가 대미지 극대화 |
| Forest 독 딜러 | 중독 디버프 + 아테나 얼티밋 시너지 |
| Light 힐러/서포터 | 아테나 디버프 제거로 힐러 보호, 연속 공격 환경 조성 |

### 카운터 관계

| 아테나가 강한 상대 | 이유 |
|-------------------|------|
| 버프 스택형 파티 | 패시브 기절 시 버프 전체 제거 |
| Dark 버프 누적 딜러 | 얼티밋 + 패시브로 버프 차단 |

| 아테나의 약점 | 이유 |
|--------------|------|
| Water 속성 파티 | 광속성 불리 없음이나 Water CC 캐릭터의 기절로 선제 제압 가능 |
| 빠른 단일 타겟 암살 | DEF 400, HP 4000으로 집중 사격에 취약 |

## NC02: 포세이돈 (Poseidon)

### 기본 정보

| 항목   | 값                    |
|--------|-----------------------|
| 속성   | Water (수)            |
| 직업   | Defender              |
| 성급   | 4★                   |
| ATK    | 560                   |
| DEF    | 900                   |
| HP     | 7200                  |
| SPD    | 76                    |
| 행동간격 | 3.95 (300/76)       |

### 스킬 상세

**[노멀] 해류 타격**
- 효과: 적 1명에게 ATK × 1.0 수속성 대미지

**[액티브] 해류의 방벽 (강화 전)**
- 효과: 자신 DEF +30% (2턴) + 아군 전체 SPD +10 (2턴) + 자신에게 도발 (전열 대상 고정, 2턴)

**[액티브] 해류의 방벽 +1**
- 효과: 자신 DEF +40% (2턴) + 아군 전체 SPD +10 (2턴) + 자신에게 도발 + 자신에게 소형 보호막 (HP의 10%)
- 강화 추가: DEF 강화 +10%, 도발 사용 시 보호막 발동

**[액티브] 해류의 방벽 +2**
- 효과: 자신 DEF +40% (2턴) + 아군 전체 SPD +15 (2턴) + 자신에게 도발 + 아군 전체 소형 보호막 (HP의 8%)
- 강화 추가: SPD 버프 +5, 보호막 아군 전체로 확장

**[얼티밋] 심해의 파도 (강화 전)**
- 효과: 적 전체에게 DEF × 0.8 수속성 대미지 + 적 전체 SPD -15 (2턴) + 자신에게 보호막 (HP의 20%)

**[얼티밋] 심해의 파도 +1**
- 효과: 적 전체에게 DEF × 1.0 수속성 대미지 + 적 전체 SPD -15 (2턴) + 자신에게 보호막 (HP의 25%)
- 강화 추가: 대미지 배율 증가, 보호막 증가

**[얼티밋] 심해의 파도 +2**
- 효과: 적 전체에게 DEF × 1.0 수속성 대미지 + 적 전체 SPD -15 (2턴) + 자신에게 보호막 (HP의 25%) + 도발 중일 때 자신이 받는 피해 -20% (즉발)
- 강화 추가: 도발 중 피해 감쇄 기믹 추가

**[패시브] 해왕의 갑주**
- 효과: 포세이돈이 피격될 때, 자신 SPD의 30%만큼 DEF 추가 증가 (2턴 유지, 쿨타임 2턴)
- SPD 76 기준: 피격 시 DEF +22.8 (소수점 버림 → +22)

### 설계 의도

Water/Defender 풀은 엘리시온(3★)과 데이노(1★)로만 구성되어 있어 수속성 파티의 내구선이 약했다. 포세이돈은 4★ 최고 등급 수탱으로 방어력 기반 공격, 아군 속도 지원, 적 속도 감소를 동시에 수행하는 다기능 탱커다.

### 메타 영향 분석

- **긍정적**: Water 파티 생존성 대폭 향상. CTB 조작(아군 +10~+15 SPD, 적 -15 SPD)으로 행동 순서를 지배.
- **주의점**: DEF 기반 대미지는 얼티밋 기준 900 × 1.0 = 900 (단독). 낮은 수치이나 생존/CC 역할이 주라 허용 범위.
- **시너지 위험**: 아군 SPD +15 + 포세이돈 패시브 DEF 증가 조합이 자기강화 루프를 만들 수 있음. 쿨타임 2턴 패시브로 제어됨.

### 시너지 파트너

| 캐릭터 유형 | 시너지 이유 |
|-------------|-------------|
| Water 고딜 마법사 | 포세이돈 SPD 버프 + 적 속도 감소 → 안전한 딜 환경 |
| 전체 힐러 | 포세이돈 도발로 집중사격 방어, 힐러가 아군 생존 집중 |
| 속도형 서포터 | 포세이돈 SPD 버프와 중복 → 아군 선행 행동 강화 |

### 카운터 관계

| 포세이돈이 강한 상대 | 이유 |
|---------------------|------|
| 속도형 딜 파티 | SPD 감소로 선공 박탈, 도발로 화력 분산 |
| Fire 계열 | Water 속성 유리 +30% |

| 포세이돈의 약점 | 이유 |
|----------------|------|
| Forest 속성 파티 | 속성 불리 -30%, 방어력 무시 기술에 취약 |
| 버프 제거 캐릭터 | 도발/DEF 버프 제거 시 생존력 급락 |

## NC03: 가이아 (Gaia)

### 기본 정보

| 항목   | 값                    |
|--------|-----------------------|
| 속성   | Forest (자연)         |
| 직업   | Attacker              |
| 성급   | 3.5★                  |
| ATK    | 900                   |
| DEF    | 420                   |
| HP     | 4200                  |
| SPD    | 85                    |
| 행동간격 | 3.53 (300/85)       |

### 스킬 상세

**[노멀] 가시 타격**
- 효과: 적 1명에게 ATK × 1.0 자연속성 대미지

**[액티브] 대지의 구속 (강화 전)**
- 효과: 적 1명에게 ATK × 2.2 자연속성 대미지 + 중독 2스택 부여 (매 턴 ATK×0.3, 3턴) + 넉백

**[액티브] 대지의 구속 +1**
- 효과: 적 1명에게 ATK × 2.4 자연속성 대미지 + 중독 3스택 부여 + 넉백
- 강화 추가: 대미지 배율 +0.2, 중독 스택 +1

**[액티브] 대지의 구속 +2**
- 효과: 적 1명에게 ATK × 2.4 자연속성 대미지 + 중독 3스택 부여 + 넉백 + 넉백 후 해당 대상 SPD -10 (1턴)
- 강화 추가: 넉백 시 속도 감소 기믹 추가

**[얼티밋] 자연의 심판 (강화 전)**
- 효과: 적 전체에게 ATK × 4.0 자연속성 대미지 + 중독 보유 적에게 방어력 무시 추가 대미지 (ATK × 0.5) + 전열 적 넉백

**[얼티밋] 자연의 심판 +1**
- 효과: 적 전체에게 ATK × 4.0 자연속성 대미지 + 중독 보유 적에게 방어력 무시 추가 대미지 (ATK × 0.8) + 전열 적 넉백
- 강화 추가: 방어력 무시 추가 대미지 +0.3

**[얼티밋] 자연의 심판 +2**
- 효과: 적 전체에게 ATK × 4.0 자연속성 대미지 + 중독 보유 적에게 방어력 무시 추가 대미지 (ATK × 0.8) + 전열 적 넉백 + 중독 스택 3이상인 적: 즉시 중독 폭발 (스택당 ATK × 0.5 추가)
- 강화 추가: 고스택 중독 폭발 기믹 추가

**[패시브] 자연의 순환**
- 효과: 가이아가 중독 상태의 적을 처치할 때, 그 적 주변의 적 1명에게 중독 전이 (남은 스택, 1턴 지속) + 자신 DEF +15% (2턴)

### 설계 의도

Forest/Attacker는 넉백 위주 단발성 전략이 전부였다. 가이아는 중독 축적 + 넉백을 결합한 컨트롤 딜러로, 누적 딜과 즉발 딜을 동시에 운용하는 새로운 자연 전략 축을 형성한다.

### 메타 영향 분석

- **긍정적**: Forest 파티에 지속 딜 레이어 추가. 중독 생태계 형성으로 기존 넉백 딜러와 역할 분담 가능.
- **주의점**: 중독 폭발(+2 강화) + 방어력 무시 조합은 DEFENDER에게 매우 강함. 고DEF 캐릭터 상대 밸런스 모니터링 필요.

### 시너지 파트너

| 캐릭터 유형 | 시너지 이유 |
|-------------|-------------|
| Forest 넉백 딜러 | 넉백으로 후열 이동 → 가이아 액티브로 중독 + 재넉백 |
| Dark 디버퍼 | 중독 + 다른 디버프 중첩 → 아테나(NC01) 얼티밋 시너지 |
| 속도형 서포터 | 가이아 SPD 85 → 빠른 중독 축적 루프 지원 |

### 카운터 관계

| 가이아가 강한 상대 | 이유 |
|-------------------|------|
| 고DEF 탱커 | 방어력 무시 중독 추가 대미지 |
| Water 속성 | 자연 속성 유리 +30% |

| 가이아의 약점 | 이유 |
|--------------|------|
| Fire 속성 | 속성 불리 -30% |
| 디버프 면역/정화 캐릭터 | 중독 전략 무력화 |

**[얼티밋] 사신의 축복 +1**
- 효과: 적 전체에게 ATK × 4.0 암속성 대미지 + 아군 전체 HP 회복 (ATK × 1.5) + 사망한 아군 1명 HP 50%로 부활

**[얼티밋] 사신의 축복 +2**
- 효과: 적 전체에게 ATK × 4.0 암속성 대미지 + 아군 전체 HP 회복 (ATK × 1.5) + 사망한 아군 1명 HP 50%로 부활 + 부활 대상에게 1턴 무적 부여
- 강화 추가: 부활 캐릭터 1턴 무적 기믹 — 부활 직후 즉시 재처치 방지

**[패시브] 사신의 계약**
- 효과: 적이 사망할 때마다 아군 전체 HP 10% 회복 + 헬의 다음 공격 치명타 확정 (중복 불가, 최근 처치 1회 적용)

### 설계 의도

Dark 속성 힐러는 기존에 없었다. 헬은 처치 연계 힐러로, 전투가 길어질수록 강해지는 역전 특화 설계다. 부활 기능은 현재 메타에 없는 유일한 메커니즘이다.

### 메타 영향 분석

- **긍정적**: 부활 메커니즘 도입으로 장기전 파티 생존성 혁신. PvE 고난이도 컨텐츠에서 핵심 유닛.
- **주의점**: 부활 + 패시브 전체 힐 + 치명타 조합은 장기 PvP에서 게임을 질질 끌 수 있음. PvP 환경에서 부활 HP 비율 하향(40% → 30%) 또는 쿨타임 제한 고려.

## NC05: 이프리트 (Ifrit)

| HP     | 3900                  |
| SPD    | 90                    |

### 스킬 상세

**[노멀] 화염 방사**
- 효과: 적 1명에게 ATK × 1.0 화속성 대미지

**[액티브] 용암 표식 (강화 전)**
- 효과: 적 1명에게 ATK × 2.2 화속성 대미지 + 해당 적에게 "용암 표식" 1스택 부여 (최대 5스택, 영구 지속, 스택당 받는 대미지 +5%)

**[액티브] 용암 표식 +1**
- 효과: 적 1명에게 ATK × 2.4 화속성 대미지 + 용암 표식 2스택 부여
- 강화 추가: 표식 부여량 2배

**[액티브] 용암 표식 +2**
- 효과: 적 1명에게 ATK × 2.4 화속성 대미지 + 용암 표식 2스택 부여 + 표식 3스택 이상인 적: 이 공격에 ATK × 0.5 추가 대미지
- 강화 추가: 표식 임계값 초과 시 즉발 추가 대미지

**[얼티밋] 용암의 심판 (강화 전)**
- 효과: 적 1명에게 ATK × 4.0 화속성 대미지 + 용암 표식 수 × 50% 대미지 증가 + 표식 전부 소모
- 5스택 기준: ATK × 4.0 × (1 + 5×0.5) = ATK × 14.0

**[얼티밋] 용암의 심판 +1**
- 효과: 적 1명에게 ATK × 4.5 화속성 대미지 + 용암 표식 수 × 60% 대미지 증가 + 표식 전부 소모
- 5스택 기준: ATK × 4.5 × (1 + 5×0.6) = ATK × 18.0
- 강화 추가: 기본 배율 +0.5, 표식 증폭 +10%

**[얼티밋] 용암의 심판 +2**
- 효과: 적 1명에게 ATK × 4.5 화속성 대미지 + 용암 표식 수 × 60% 대미지 증가 + 표식 전부 소모 + 소모된 표식 수만큼 주변 적에게 ATK × 0.3 화염 튀김 대미지
- 강화 추가: 표식 소모 시 범위 스플래시 추가

**[패시브] 집착의 불꽃**
- 효과: 이프리트가 같은 적을 연속으로 공격할 때마다 대미지 +15% 누적 (최대 5중첩, 대상 변경 시 즉시 초기화)

### PvE 보스킬러 성능 계산

**풀스택 기준 (용암 표식 5스택 + 패시브 5중첩, 강화 +1 얼티밋)**

```
기본 배율: ATK × 4.5
표식 증폭: × (1 + 5 × 0.6) = × 4.0
패시브 누적: × (1 + 5 × 0.15) = × 1.75
최종 대미지: 980 × 4.5 × 4.0 × 1.75 = 30,870
```

> **경고**: 풀스택 시 단일 대미지 30,870은 이상치 구간. PvP 환경에서는 표식 초기화(버프 제거, 대상 변경 강제) 메커니즘으로 제어. PvE 보스전 전용 설계 컨셉이므로 허용되나 PvP 대미지 캡 적용 검토 필요.

### 설계 의도

PvE 보스 공략에 특화된 캐릭터가 없어 장기 보스전 콘텐츠 동기가 약했다. 이프리트는 단일 대상에 표식을 누적하고 터뜨리는 보스킬러로, PvP에서는 대상 전환으로 약점이 있어 양 모드 밸런스를 자연스럽게 분리한다.

### 메타 영향 분석

- **긍정적**: PvE 보스 레이드 콘텐츠에서 필수 유닛 등장. Fire 파티 딜 상한선 대폭 상승.
- **주의점**: PvP에서 풀스택 달성 시 원턴킬 가능. 표식 누적을 방해하는 메커니즘(CC, 넉백으로 이프리트 행동 차단, 대상 보호막 등) 이 자연스러운 카운터로 작동하는지 검증 필요.

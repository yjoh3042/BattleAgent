### 🏟️ 공격덱 / 방어덱 타임오버 룰

```
┌──────────┬────────────────────────────────────────────┐
│ 덱 타입   │ 타임오버 시 결과                             │
├──────────┼────────────────────────────────────────────┤
│ 공격덱    │ 패배 (적 전멸 실패 = 공격 목표 미달성)       │
│ (OFFENSE) │ → 제한 시간 내 적을 처치해야 승리            │
│          │ → 고배율 딜러 확보가 편성의 필수 조건          │
├──────────┼────────────────────────────────────────────┤
│ 방어덱    │ 승리 (생존 성공 = 방어 목표 달성)            │
│ (DEFENSE) │ → 제한 시간 동안 살아남으면 승리             │
│          │ → 탱커+힐러 중심 생존형이 절대 유리            │
└──────────┴────────────────────────────────────────────┘
기본값: OFFENSE (공격덱) — 기존 동작과 동일.
```

> **전략적 함의**: 방어덱은 적의 화력을 버티기만 하면 되므로 탱커/힐러 중심 구성이 압도적.
> 공격덱은 200턴/300.0시간 내에 적을 전멸시켜야 하므로 고배율 딜러가 필수.

### 속성 상성 배율

```
화(Fire) → 목(Forest) : 1.2배 (+20%)
목(Forest) → 수(Water) : 1.2배 (+20%)
수(Water) → 화(Fire) : 1.2배 (+20%)
광(Light) ↔ 암(Dark) : 상호 1.2배 (+20%)
동일/무관 속성 : 1.0배
역상성 : 0.8배 (-20%)
```

### CTB 턴 시스템
- `action_time = 300 / SPD` → SPD 높을수록 먼저 행동
- 라운드 = 10 time unit마다 리셋, SP 풀 초기화
- SP: 턴마다 +1 (아군/적 공유 풀), 최대 10
- 궁극기: SP 소비 + 4턴 쿨다운, **궁극기 + 일반/액티브 동시 사용 가능 (보너스 액션)**
- 전투 제한: 최대 200턴 / 300.0 time → **초과 시 공격덱=패배, 방어덱=승리**

## ⚡ M03 — 독살처형 (Poison Execute)

> **기믹 참조**: 붕괴:스타레일 Break→Execute — 약점 파괴 후 처형 조건 달성

### 편성

| 캐릭터 | 역할/등급 | 포지션 | 핵심 기여 |
|--------|-----------|--------|----------|
| **이브** | ATK / 3★ | 처형인 | HP<30% 대상 3.00×+2.00×=5.00× 처형 |
| **바리** | MAG / 3★ | 독 딜러 | 모든 스킬에 독 부여 |
| **도계화** | MAG / 3★ | 독+스턴 | 독 + Stun 복합 |
| **그릴라** | SUP / 3★ | 버퍼 | ATK+% + CRI+% 버프 |
| **브라우니** | HLR / 3★ | 힐러 | 전체 힐 + 독 내성 지원 |

### 기믹 메커니즘
```
[독→처형 2단계]
Phase 1: 독 축적
바리 → 모든 스킬에 독 (매 스킬마다 독 DoT 추가)
도계화 → 독 + Stun (행동 방해 + DoT)

Phase 2: 처형 발동
```

## ⚖️ 5대 법칙 (Meta Rules v2.0)

### 법칙 1: 힐러 필수 법칙
```
실제 힐 스킬이 있는 캐릭터가 없으면 0%가 된다.

근거:
- 힐러 없는 팀: 상대 DoT/지속 딜에 소모됨
- 힐러 있는 팀: 브라우니가 에레보스 AoE를 버팀 (M03 vs M09)
- v5.0 모든 10개 메타: 힐러 최소 1명 필수 편성 원칙 준수
```

### 법칙 2: 처형 우선 법칙
```
HP% 기반 처형 메커닉이 DoT+버퍼 조합보다 효율적이다.

근거:
- M03(처형): 69.1% S티어
- M01(DoT): 65.7% S티어
- 처형이 DoT보다 3.4%p 높음
- 이유: 처형은 HP 조건 달성 즉시 확정 킬, DoT는 누적 시간 필요
```

### 법칙 3: 방어덱 생존 법칙
```
킬 파워가 없어도 방어덱에서는 생존만으로 승리한다.

근거: M05 철벽요새
- 공격덱 0% (킬 파워 없음)
- 방어덱 70.4% (타임오버 승리 조건)
- 방어덱 전용 운용 시 S티어급 성능 발휘
```

### 법칙 4: CC 킬파워 법칙
```
CC만으로는 이길 수 없다. 반드시 DPS 동반 필요.

v3.x 문제: 마야우엘+마프데트 CC only → 낮은 승률
v5.0 해결: Ult에 AoE 데미지 추가 → M02 승률 향상
에르제베트 6.50× → CC 쿨다운 중 딜 담당

CC 팀이 단독으로는 타임오버 내 처치 불가능
```

### 법칙 5: 상성 우위 법칙
```
어떤 1개 메타도 모든 메타를 이기지 못한다.

M03 독살처형 (69.1% S티어) 조차:
→ M04 하이퍼캐리에 0% 패배

M09 전멸폭격 (68.3% S티어) 조차:
→ M02 빙결감옥에 0% 패배

결론: 완벽한 메타는 없음. 상성 기반 전략적 픽이 핵심.
```

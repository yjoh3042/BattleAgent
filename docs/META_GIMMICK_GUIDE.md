## 📐 전투 시스템 기초 수치

### 데미지 공식
```
base = ATK × (1 - DEF/(ATK+DEF))
damage = floor(base × skillMult × elMult × critMult)
crit = 15% 확률로 1.5배
```

### 속성 상성 배율
```
화(Fire) → 목(Forest) : 1.2배 (+20%)
목(Forest) → 수(Water) : 1.2배 (+20%)
수(Water) → 화(Fire)  : 1.2배 (+20%)
광(Light) ↔ 암(Dark)  : 상호 1.2배 (하이리스크·하이리턴)
동일/무관 속성         : 1.0배
역상성               : 0.8배 (-20%)
```

### CTB 턴 시스템
- `action_time = 300 / SPD` → SPD 높을수록 먼저 행동
- SP: 턴마다 +1, 최대 10, 파티 공유 풀
- 궁극기: SP 소비 + 4턴 쿨다운, 보너스 액션 (일반/액티브와 동시 사용 가능)
- **패시브 스킬**: 4번째 스킬 슬롯, 트리거/전투시작 시 자동 발동
- 전투 제한: 최대 200턴 / 300.0 time → 초과 시 공격덱=패배, 방어덱=승리

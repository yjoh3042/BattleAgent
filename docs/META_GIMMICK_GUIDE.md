## 📐 전투 시스템 기초 수치

### 데미지 공식
```
base = ATK × (1 - DEF/(ATK+DEF))
damage = floor(base × skillMult × elMult × critMult)
crit = 15% 확률로 1.5배
```

### 속성 상성 배율
```
화(Fire) → 목(Forest) : 1.2배 (+20%)
목(Forest) → 수(Water) : 1.2배 (+20%)
수(Water) → 화(Fire)  : 1.2배 (+20%)
광(Light) ↔ 암(Dark)  : 상호 1.2배 (하이리스크·하이리턴)
동일/무관 속성         : 1.0배
역상성               : 0.8배 (-20%)
```

### CTB 턴 시스템
- `action_time = 300 / SPD` → SPD 높을수록 먼저 행동
- SP: 턴마다 +1, 최대 10, 파티 공유 풀
- 궁극기: SP 소비 + 4턴 쿨다운, 보너스 액션 (일반/액티브와 동시 사용 가능)
- 전투 제한: 최대 200턴 / 300.0 time → 초과 시 공격덱=패배, 방어덱=승리

### 주요 발견

1. **CC가 왕, 독이 견제**: CC 기반(77.4%) 최강이지만, 독 DoT(67.6%)가 유일한 자연적 카운터
2. **가위바위보 삼각 성립**: M12→M04→M02→M09→M12 순환, 완벽 메타 없음
3. **디버프면역의 가치**: M11 광속성벽은 전체 28.8%지만 M02 빙결감옥을 100% 카운터
4. **속성보다 기믹 상성이 중요**: 화→목 +20%보다 CC→DoT 기믹 카운터가 승패를 더 크게 좌우
5. **보호 기믹은 방어덱 전환 필수**: 배리어+면역은 공격덱 킬프레셔 부족, 방어덱에서 진가 발휘

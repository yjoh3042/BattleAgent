## 4. 메타별 상세 전투 플로우

### M01. 🔥 화상폭발 (Burn Explosion) — 59.7%

**로스터**: 카라라트리(화/ATK) · 다비(화/MAG) · 드미테르(화/SUP) · 살마키스(화/SUP) · 지바(화/HLR)

**핵심 기믹**: 화상 스택 → DAMAGE_BURN_BONUS 2배 폭발 → DEBUFF_SPREAD 전이

**전투 플로우**:
1. **초반 (1~3턴)**: 다비·살마키스가 적 전체에 화상 부여, 살마키스 debuff_turn_inc로 화상 지속
2. **중반 (4~6턴)**: 드미테르 ALLY_SAME_ELEMENT 화속성 전체 공/속 버프 → 카라라트리 ON_CRITICAL_HIT 패시브로 추가 화상
3. **후반 (7턴~)**: 카라라트리 얼티밋 DAMAGE_BURN_BONUS (화상 대상 2배+크리) → 처치 시 다비 DEBUFF_SPREAD 패시브로 화상 십자 전이
4. **지속력**: 지바 HEAL_PER_HIT 얼티밋으로 적 피해 적중만큼 아군 회복

**강점**: M04 버프연쇄(100%), M05 철벽요새(100%) — 화상이 보호막/버프를 무시하는 지속 피해
**약점**: M02 빙결처형(0%), M07 속도학살(0%) — 속도 우위에 화상 스택 쌓기 전 처형당함

### M02. 🧊 빙결처형 (Freeze Execute) — 38.1%

**로스터**: 이브(수/ATK) · 도계화(수/MAG) · 마야우엘(수/MAG) · 상아(수/SUP) · 에우로스(수/HLR)

**핵심 기믹**: CC 릴레이 → 행동 불가 상태에서 처형

**전투 플로우**:
1. **초반**: 상아 속도 디버프 + 방깎으로 적 템포 붕괴
2. **중반**: 도계화 STUN + 마야우엘 FREEZE + SP_LOCK CC 연쇄
3. **후반**: 이브 얼티밋으로 CC된 적 처형 + FREEZE 부여
4. **지속력**: 에우로스 ALLY_LOWEST_HP_3 (3명 동시 회복) + 정화

**강점**: M01 화상폭발(90%), M06 광전사(83%) — CC로 화상 스택/자해 루프 차단
**약점**: M05 철벽요새(0%), M07 속도학살(0%), M09 암살침투(0%) — 디버프면역이나 관통으로 CC 무시

### M03. 🌿 독확산정원 (Poison Spread) — 57.7%

**로스터**: 에레보스(목/MAG) · 판(목/MAG) · 그릴라(목/SUP) · 메티스(목/DEF) · 브라우니(목/HLR)

**핵심 기믹**: 중독/수면 DEBUFF_SPREAD + DAMAGE_ESCALATE 누적 딜

**전투 플로우**:
1. **초반**: 판 중독+수면 부여, 그릴라 ALLY_ADJACENT 치확/치피 버프
2. **중반**: 에레보스 DAMAGE_ESCALATE 노말 (사용마다 피해↑) + 처치 시 판 DEBUFF_SPREAD 전이
3. **후반**: 에레보스 ON_ULTIMATE_USED 공↑ 누적 → 얼티밋 폭발
4. **지속력**: 메티스 STAT_STEAL DEF 강탈 + 브라우니 HEAL_CURRENT_HP_SCALE 풀피 유지

**강점**: M04 버프연쇄(83%), M06 광전사(100%), M09 암살침투(100%) — DoT가 수동적 팀을 지속 소모
**약점**: M01 화상폭발(23%), M05 철벽요새(17%) — 화상 폭발에 밀리거나 디버프면역에 막힘

**로스터**: 아슈토레스(광/ATK) · 루미나(광/MAG) · 시트리(광/SUP) · 엘리시온(수/DEF) · 티와즈(광/HLR)

**전투 플로우**:
1. **초반**: 시트리 ALLY_ADJACENT ATK+CRI 버프 → 루미나/아슈토레스 ON_BUFF_GAINED 치확·치피↑
2. **중반**: 루미나 추가 버프 → ON_BUFF_GAINED 재발동 → 스택 누적
3. **링크**: 엘리시온 LINK_BUFF로 뒤칸 아군에게 버프 공유
4. **폭발**: 아슈토레스 고치확·고치피 상태로 딜 폭발
5. **보호**: 티와즈 ALLY_LOWEST_HP_3 회복 + 디버프면역

**강점**: M02 빙결처형(60%), M08 크리연쇄(90%), M10 성속결속(87%) — 버프 스케일링이 순수 딜팀 압도

4. **딜**: 아슈토레스 고치확 상태 폭발 딜
5. **회복**: 티와즈 ALLY_LOWEST_HP_3 (3명 동시) + 디버프면역

**강점**: M02 빙결처형(93%), M09 암살침투(100%) — 디버프면역+보호막이 CC/관통 방어
